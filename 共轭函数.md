# 共轭函数 (Fenchel Conjugate)

Fenchel共轭（或称Legendre-Fenchel变换）是[[凸函数]]理论中最重要的对偶构造，它将函数 $f$ 映射为其共轭 $f^*$，恒为凸函数。

[[凸函数]] | [[对偶理论]] | [[保凸运算与复合规则]]

---

## 一、定义

函数 $f: \mathbb{R}^n \to \mathbb{R} \cup \{+\infty\}$ 的**Fenchel共轭**：

$$\boxed{f^*(\mathbf{y}) = \sup_{\mathbf{x} \in \text{dom }f} \left(\mathbf{y}^T \mathbf{x} - f(\mathbf{x})\right)}$$

### 1.1 几何解释

$f^*(\mathbf{y})$ = 斜率为 $\mathbf{y}$ 的**仿射函数** $\mathbf{y}^T \mathbf{x} - \alpha$ 中，与 $f$ 的上镜图 $\text{epi }f$ 有最大截距的 $\alpha$ 值。

等价：$-f^*(\mathbf{y})$ = 斜率为 $\mathbf{y}$ 的直线在 $f$ 的图像**之下**能取的最大截距。

### 1.2 共轭的关键性质

**$f^*$ 恒为凸函数**（即使 $f$ 非凸），因为 $f^*$ 是一族关于 $\mathbf{y}$ 的**仿射函数**的逐点上确界。

---

## 二、Fenchel不等式

### 2.1 基本形式

由共轭的定义直接得到：

$$\boxed{\mathbf{x}^T \mathbf{y} \leq f(\mathbf{x}) + f^*(\mathbf{y}), \quad \forall \mathbf{x}, \mathbf{y}}$$

这称为**Fenchel不等式**（或称Young-Fenchel不等式）。

### 2.2 经典Young不等式的推广

$f(x) = \frac{1}{p}|x|^p$（$p > 1$）的共轭：$f^*(y) = \frac{1}{q}|y|^q$（$\frac{1}{p} + \frac{1}{q} = 1$）

Fenchel不等式给出：

$$xy \leq \frac{|x|^p}{p} + \frac{|y|^q}{q}$$

即经典的Young不等式。

---

## 三、二次共轭

### 3.1 二次共轭

$$f^{**}(\mathbf{x}) = \sup_{\mathbf{y}} \left(\mathbf{x}^T \mathbf{y} - f^*(\mathbf{y})\right)$$

### 3.2 基本定理

$$\boxed{f^{**} \leq f}$$

且等号成立（$f^{**} = f$）**当且仅当** $f$ 为**闭凸函数**。

**推论**：对任意函数 $f$，$f^{**}$ 是 $f$ 的**最大闭凸下界**（convex envelope / convex hull）。

> $f^{**}$ = $f$ 的凸包络（取 $\text{conv}(\text{epi }f)$ 对应的函数）。

---

## 四、常见共轭对

### 4.1 经典共轭对

| $f(\mathbf{x})$ | $\text{dom }f$ | $f^*(\mathbf{y})$ | $\text{dom }f^*$ |
|----------------|----------------|------------------|------------------|
| $\frac{1}{2}\|\mathbf{x}\|_2^2$ | $\mathbb{R}^n$ | $\frac{1}{2}\|\mathbf{y}\|_2^2$ | $\mathbb{R}^n$ |
| $\|\mathbf{x}\|$ | $\mathbb{R}^n$ | $I_{\{\|\mathbf{y}\|_* \leq 1\}}$ | $\|\mathbf{y}\|_* \leq 1$ |
| $\|\mathbf{x}\|_2$ | $\mathbb{R}^n$ | $I_{\{\|\mathbf{y}\|_2 \leq 1\}}$ | $\|\mathbf{y}\|_2 \leq 1$ |
| $\|\mathbf{x}\|_1$ | $\mathbb{R}^n$ | $I_{\{\|\mathbf{y}\|_\infty \leq 1\}}$ | $\|\mathbf{y}\|_\infty \leq 1$ |
| $e^x$ | $\mathbb{R}$ | $y\log y - y$ | $y > 0$ |
| $-\log x$ | $x > 0$ | $-\log(-y)-1$ | $y < 0$ |
| $x\log x$ | $x > 0$ | $e^{y-1}$ | $\mathbb{R}$ |
| $\frac{1}{p}\|\mathbf{x}\|_p^p$（$p>1$） | $\mathbb{R}^n$ | $\frac{1}{q}\|\mathbf{y}\|_q^q$（$1/p+1/q=1$） | $\mathbb{R}^n$ |
| $\log\sum e^{x_i}$ | $\mathbb{R}^n$ | $\sum y_i\log y_i$（$\sum y_i=1,y_i \geq 0$） | 概率单纯形 |
| $-\log\det\mathbf{X}$ | $\mathbf{S}_{++}^n$ | $-\log\det(-\mathbf{Y})-n$ | $-\mathbf{Y} \succ 0$ |

其中 $I_{S}$ 是集合 $S$ 的指示函数（在 $S$ 上为 $0$，否则 $+\infty$）。

### 4.2 指示函数的共轭

凸集 $C$ 的指示函数 $I_C$ 的共轭是 $C$ 的**支撑函数**：

$$I_C^*(\mathbf{y}) = \sup_{\mathbf{x} \in C} \mathbf{y}^T \mathbf{x} = s_C(\mathbf{y})$$

## 五、共轭的运算性质

### 5.1 表

| 原函数 | 共轭函数 |
|--------|---------|
| $f(\mathbf{x}) + a$ | $f^*(\mathbf{y}) - a$ |
| $a f(\mathbf{x})$（$a > 0$） | $a f^*(\mathbf{y}/a)$ |
| $f(\mathbf{x}) + \mathbf{a}^T\mathbf{x} + b$ | $f^*(\mathbf{y} - \mathbf{a}) - b$ |
| $f(\mathbf{A}\mathbf{x})$（$\mathbf{A}$ 可逆） | $f^*(\mathbf{A}^{-T}\mathbf{y})$ |
| 可分和 $\sum f_i(x_i)$ | $\sum f_i^*(y_i)$ |
| $(f_1 \square f_2)(\mathbf{x}) = \inf_{\mathbf{u}+\mathbf{v}=\mathbf{x}} (f_1(\mathbf{u}) + f_2(\mathbf{v}))$ | $f_1^*(\mathbf{y}) + f_2^*(\mathbf{y})$ |

其中 $f_1 \square f_2$ 称为**下卷积**（infimal convolution）。

### 5.2 梯度与共轭的逆关系

若 $f$ 严格凸且闭，则：

$$\mathbf{y} = \nabla f(\mathbf{x}) \iff \mathbf{x} = \nabla f^*(\mathbf{y})$$

即 $\nabla f^* = (\nabla f)^{-1}$。

## 六、共轭与对偶理论

Fenchel共轭是[[对偶理论|Lagrange对偶]]的推广。

### 6.1 Fenchel对偶定理

$$\min_{\mathbf{x}} \{f(\mathbf{x}) + g(\mathbf{x})\} \quad \text{的对偶:} \quad \max_{\mathbf{y}} \{-f^*(\mathbf{y}) - g^*(-\mathbf{y})\}$$

最优性条件：$\mathbf{0} \in \partial f(\mathbf{x}^*) + \partial g(\mathbf{x}^*)$ ⇔ $\mathbf{x}^* \in \partial f^*(\mathbf{y}^*) \cap (-\partial g^*(-\mathbf{y}^*))$。

### 6.2 Lagrange对偶的Fenchel视角

带约束的优化问题可以通过指示函数重写为无约束形式，然后应用Fenchel对偶 ⇒ 得到Lagrange对偶。

---

## 七、Moreau包络与近端算子

### 7.1 Moreau包络

函数 $f$ 的 **Moreau包络**（Moreau Envelope）：

$$M_f(\mathbf{x}) = \inf_{\mathbf{y}} \left(f(\mathbf{y}) + \frac{1}{2}\|\mathbf{y} - \mathbf{x}\|_2^2\right)$$

$M_f$ 是 $f$ 与 $\frac{1}{2}\|\cdot\|^2$ 的下卷积：$M_f = f \square \frac{1}{2}\|\cdot\|^2$

$M_f$ 是**光滑**凸函数（梯度为 $1$-Lipschitz），即使 $f$ 非光滑。

### 7.2 与共轭的关系

$$M_f(\mathbf{x}) + M_{f^*}(\mathbf{x}) = \frac{1}{2}\|\mathbf{x}\|^2$$

（Moreau分解）

### 7.3 近端算子

[[近端梯度法与ADMM|近端算子]] 与 Moreau包络相关：

$$\text{prox}_f(\mathbf{v}) = \arg\min_{\mathbf{x}} \left(f(\mathbf{x}) + \frac{1}{2}\|\mathbf{x} - \mathbf{v}\|_2^2\right)$$
