## 仿射集

### 定义

集合 $C \subseteq \mathbb{R}^n$ 是**仿射集**，若：

$$\forall \mathbf{x}_1, \mathbf{x}_2 \in C, \forall \theta \in \mathbb{R}: \quad \theta \mathbf{x}_1 + (1-\theta)\mathbf{x}_2 \in C$$

等价：$C$ 包含通过其中任意两点的**完整直线**。

### 仿射组合

$\mathbf{x}_1, \ldots, \mathbf{x}_k$ 的**仿射组合**是 $\sum_{i=1}^{k} \theta_i \mathbf{x}_i$，其中 $\sum \theta_i = 1$。

- 仿射集 = 对仿射组合封闭的集合
- 仿射包 $\text{aff }S$ = $S$ 中点的所有仿射组合的集合

### 与子空间的关系

任意仿射集 $C$ 可表示为 $C = V + \mathbf{x}_0$，其中 $V$ 是子空间，$\mathbf{x}_0 \in C$。

等价：$\{\mathbf{x} \mid \mathbf{A}\mathbf{x} = \mathbf{b}\}$ 是仿射集。

## 凸集

### 定义

$$\forall \mathbf{x}_1, \mathbf{x}_2 \in C, \forall \theta \in [0,1]: \quad \theta \mathbf{x}_1 + (1-\theta)\mathbf{x}_2 \in C$$

### 凸组合

$\sum_{i=1}^{k} \theta_i \mathbf{x}_i$ 称为**凸组合**，若 $\theta_i \geq 0$ 且 $\sum \theta_i = 1$。

- 凸集 = 对凸组合封闭的集合
- 凸包 $\text{conv }S$ = $S$ 中点的所有凸组合的集合

### 凸包的刻画

$$\text{conv }S = \left\{\sum_{i=1}^{k} \theta_i \mathbf{x}_i \mid \mathbf{x}_i \in S, \theta_i \geq 0, \sum_{i=1}^{k} \theta_i = 1, k \in \mathbb{N}\right\}$$

### Carathéodory定理

设 $S \subseteq \mathbb{R}^n$。$\text{conv }S$ 中任一点可表示为 $S$ 中**至多 $n+1$** 个点的凸组合。

> 维数 $n$ 提供了凸组合所需点数的最紧上界。

## 凸锥

### 锥

$$\forall \mathbf{x} \in C, \forall \theta \geq 0: \quad \theta \mathbf{x} \in C$$

### 凸锥

$$\forall \mathbf{x}_1, \mathbf{x}_2 \in C, \forall \theta_1, \theta_2 \geq 0: \quad \theta_1 \mathbf{x}_1 + \theta_2 \mathbf{x}_2 \in C$$

等价：$C$ 对**非负线性组合**封闭。

### 锥组合与锥包

- **锥组合**：$\sum \theta_i \mathbf{x}_i$，$\theta_i \geq 0$（不要求和=1）
- **锥包** $\text{cone }S$：$S$ 中点的所有锥组合

---

## 包含关系图

```
    子空间 ⊂ 仿射集 ⊂ 凸集
                ∪
              凸锥  ⊂ 凸集
```

- 子空间 = 包含原点的仿射集
- 仿射集 = 凸 + 线性（$\mathbb{R}$ 上的 $\theta$ 而非仅 $[0,1]$）
- 凸锥 = 凸 + 锥

## 重要凸集的详细性质

### 超平面与半空间

$$\mathcal{H} = \{\mathbf{x} \mid \mathbf{a}^T \mathbf{x} = b\} \quad \text{（超平面，仿射）}$$
$$\mathcal{H}_- = \{\mathbf{x} \mid \mathbf{a}^T \mathbf{x} \leq b\} \quad \text{（闭半空间，凸）}$$
$$\mathcal{H}_{++} = \{\mathbf{x} \mid \mathbf{a}^T \mathbf{x} < b\} \quad \text{（开半空间，凸）}$$

超平面由法向量 $\mathbf{a}$ 和偏移 $b/\|\mathbf{a}\|_2$ 决定。

### Euclid球

$$B(\mathbf{x}_c, r) = \{\mathbf{x} \mid \|\mathbf{x} - \mathbf{x}_c\|_2 \leq r\} = \{\mathbf{x}_c + r\mathbf{u} \mid \|\mathbf{u}\|_2 \leq 1\}$$

凸性源于三角不等式。

### 椭球

$$\mathcal{E} = \{\mathbf{x} \mid (\mathbf{x} - \mathbf{x}_c)^T \mathbf{P}^{-1} (\mathbf{x} - \mathbf{x}_c) \leq 1\}$$

其中 $\mathbf{P} \in \mathbf{S}_{++}^n$。椭球是 Euclid 球在仿射变换下的像。

另一种表示：$\mathcal{E} = \{\mathbf{x}_c + \mathbf{A}\mathbf{u} \mid \|\mathbf{u}\|_2 \leq 1\}$，$\mathbf{A} = \mathbf{P}^{1/2}$。

### 多面体 (Polyhedron)

$$\mathcal{P} = \{\mathbf{x} \mid \mathbf{A}\mathbf{x} \preceq \mathbf{b}, \mathbf{C}\mathbf{x} = \mathbf{d}\} = \{\mathbf{x} \mid \mathbf{a}_i^T \mathbf{x} \leq b_i, i=1,\ldots,m, \mathbf{C}\mathbf{x}=\mathbf{d}\}$$

多面体 = 有限个半空间和超平面的交集。

**多胞体 (Polytope)** = 有界多面体 = 有限点集的凸包。

### 范数球与范数锥

- **范数球**：$\{\mathbf{x} \mid \|\mathbf{x} - \mathbf{x}_c\| \leq r\}$（任意范数，凸性来自三角不等式与齐次性）
- **范数锥**：$\{(\mathbf{x}, t) \mid \|\mathbf{x}\| \leq t\} \subseteq \mathbb{R}^{n+1}$（二阶锥是其特例）

### 半正定锥

$$\mathbf{S}_+^n = \{\mathbf{X} \in \mathbf{S}^n \mid \mathbf{X} \succeq 0\}$$

- 是 $\mathbf{S}^n$（$n \times n$ 对称矩阵空间）中的**真锥**
- $\mathbf{S}_{++}^n = \{\mathbf{X} \in \mathbf{S}^n \mid \mathbf{X} \succ 0\}$ 是其内部

## 凸集的拓扑性质

### 闭包与内部

- 凸集的**闭包** $\text{cl }C$ 仍为凸集
- 凸集的**内部** $\text{int }C$ 仍为凸集
- 凸集的**相对内部** $\text{relint }C$ 非空（若 $C \neq \varnothing$）

### 相对内部与相对边界

- $\text{relint }C$：在 $\text{aff }C$ 的诱导拓扑中的内部
- 例如：三维空间中一条线段，其 $\text{int }C = \varnothing$，但 $\text{relint }C \neq \varnothing$

### 凸集的维数

$\dim C = \dim(\text{aff }C)$（仿射包的维数）
