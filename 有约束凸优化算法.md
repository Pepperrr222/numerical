# 有约束凸优化算法

考虑[[凸优化问题标准形式|凸优化问题]]：

$$\min_{\mathbf{x}} f_0(\mathbf{x}) \quad \text{s.t.} \quad f_i(\mathbf{x}) \leq 0, \mathbf{A}\mathbf{x} = \mathbf{b}$$

有约束优化算法将约束"内化"以便应用[[无约束凸优化算法]]的技术。

## 一、投影梯度法 (Projected Gradient Descent)

### 1.1 基本形式

$$\mathbf{x}^{(k+1)} = P_\mathcal{C}\left(\mathbf{x}^{(k)} - t_k \nabla f(\mathbf{x}^{(k)})\right)$$

其中 $P_\mathcal{C}(\mathbf{y}) = \arg\min_{\mathbf{x} \in \mathcal{C}} \|\mathbf{x} - \mathbf{y}\|_2$ 是向凸集 $\mathcal{C}$ 的**Euclid投影**。

### 1.2 关键条件

投影计算必须是**闭式解或可高效计算**的。

常见易投影的集合：

| 集合 | 投影 |
|------|------|
| 非负卦限 $\mathbb{R}_+^n$ | $[\max(0, x_i)]$ |
| Euclid球 | 缩放至球面 |
| 仿射空间 $\mathbf{A}\mathbf{x}=\mathbf{b}$ | 线性投影 |
| 半正定锥 $\mathbf{S}_+^k$ | 特征值阈值化 |
| $\ell_1$ 球 | 软阈值投影 |
| Box约束 $l \leq x \leq u$ | 逐元素clip |

### 1.3 收敛率

与梯度下降相同：$O(1/k)$（光滑凸）；$O((1 - \mu/L)^k)$（光滑强凸）。

## 二、障碍法 (Barrier Method)

### 2.1 对数障碍函数

$$\phi(\mathbf{x}) = -\sum_{i=1}^{m} \log(-f_i(\mathbf{x})), \quad \text{dom }\phi = \{\mathbf{x} \mid f_i(\mathbf{x}) < 0\}$$

$\phi$ 是凸函数，在可行域边界处趋于 $+\infty$，形成"障碍墙"。

### 2.2 障碍问题

$$\min_{\mathbf{x}} \quad t f_0(\mathbf{x}) + \phi(\mathbf{x}) \quad \text{s.t.} \quad \mathbf{A}\mathbf{x} = \mathbf{b}$$

- $t > 0$ 为精度参数
- $t$ 大 → 逼近原问题；$t$ 小 → $\phi$ 主导（可行性优先）

### 2.3 中心路径

$$\mathbf{x}^*(t) = \arg\min_{\mathbf{x}: \mathbf{A}\mathbf{x}=\mathbf{b}} \left(t f_0(\mathbf{x}) + \phi(\mathbf{x})\right)$$

$\{\mathbf{x}^*(t) \mid t > 0\}$ 定义为**中心路径**。

性质：$\lim_{t \to \infty} \mathbf{x}^*(t) = \mathbf{x}^*$（原问题最优解）。

### 2.4 障碍法框架

```
初始化: t^(0) > 0, μ > 1, 容差 ε > 0, 可行初始点 x
while m/t < ε:
    中心步: 从 x 开始，用Newton法计算 x^*(t)
        (求解 min t·f₀ + φ, s.t. Ax = b)
    x ← x^*(t)
    t ← μ·t
```

### 2.5 复杂度

外层迭代次数：$O(\sqrt{m} \log(m/(t^{(0)}\epsilon)))$（若使用自和谐障碍）。
每步Newton迭代：约20-30次（二次收敛）。

总复杂度：$O(\sqrt{m} \log(1/\epsilon))$ 次Newton迭代。

### 2.6 自和谐性 (Self-Concordance)

障碍函数 $\phi$ 为**自和谐**函数若满足：

$$|f'''(x)| \leq 2 f''(x)^{3/2}$$

这保证了Newton法在障碍子问题上的**二次收敛**且步长可用回溯线搜索确定。

对数障碍 $-\log(-f_i)$ 对线性约束和二阶锥约束是自和谐的；半正定锥的 $-\log\det$ 也是自和谐的。

## 三、原始-对偶内点法 (Primal-Dual IPM)

### 3.1 扰动KKT条件

从KKT条件出发，将互补松弛性 $\lambda_i f_i = 0$ 扰动为：

$$\lambda_i f_i(\mathbf{x}) = -\frac{1}{t}, \quad i = 1, \ldots, m$$

扰动KKT系统：

$$\begin{aligned} \nabla f_0(\mathbf{x}) + \sum_{i=1}^{m} \lambda_i \nabla f_i(\mathbf{x}) + \mathbf{A}^T \boldsymbol{\nu} &= \mathbf{0} \\ f_i(\mathbf{x}) + s_i = 0, \quad s_i &\geq 0 \\ \lambda_i s_i &= \frac{1}{t} \\ \mathbf{A}\mathbf{x} &= \mathbf{b} \end{aligned}$$

### 3.2 Newton步

对扰动KKT系统直接应用一步Newton法（而非完全求解子问题），每次迭代同时更新原始和对偶变量。

### 3.3 优势

- 比障碍法更快（不需要在每个 $t$ 精确求解子问题）
- 自动产生对偶变量
- 实际应用中最流行的凸优化求解方法

## 四、增广Lagrange法 (Augmented Lagrangian / Method of Multipliers)

### 4.1 增广Lagrange函数

$$L_\rho(\mathbf{x}, \boldsymbol{\lambda}, \boldsymbol{\nu}) = f_0(\mathbf{x}) + \sum_{i=1}^{m} \lambda_i f_i(\mathbf{x}) + \sum_{i=1}^{p} \nu_i h_i(\mathbf{x}) + \frac{\rho}{2}\sum_{i=1}^{m} \max(0, f_i(\mathbf{x}))^2 + \frac{\rho}{2}\sum_{i=1}^{p} h_i(\mathbf{x})^2$$

### 4.2 迭代

1. 固定 $(\boldsymbol{\lambda}^{(k)}, \boldsymbol{\nu}^{(k)})$，最小化 $L_\rho(\mathbf{x}, \boldsymbol{\lambda}^{(k)}, \boldsymbol{\nu}^{(k)})$（用无约束方法）
2. 更新乘子：$\lambda_i^{(k+1)} = \max(0, \lambda_i^{(k)} + \rho f_i(\mathbf{x}^{(k+1)}))$，$\nu_i^{(k+1)} = \nu_i^{(k)} + \rho h_i(\mathbf{x}^{(k+1)})$

这是ADMM的基础，见 [[近端梯度法与ADMM]]。

---

## 五、Active-Set方法（二次规划专用）

### 5.1 基本思想

对QP问题，猜测哪些不等式约束是"紧的"（active），求解仅含等式约束的QP子问题，然后验证KKT条件，调整active set。

### 5.2 优点与局限

- 对中小规模QP非常高效
- 不适合大规模（因为active set可能很大）
