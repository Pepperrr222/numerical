# 凸优化算法 (Convex Optimization Algorithms)

凸优化算法将理论转化为可执行的数值计算。由于[[凸优化问题]]的局部最优 = 全局最优，算法只需保证**收敛到驻点**即可。

## 算法分类总览

| 问题类型 | 主要算法 | 收敛速度 |
|---------|---------|---------|
| 无约束光滑 | 梯度下降 | $O(1/k)$ |
| 无约束光滑 | 加速梯度下降 (Nesterov) | $O(1/k^2)$ |
| 无约束光滑 | Newton法 | 二次收敛 |
| 无约束光滑 | 拟Newton法 (BFGS/L-BFGS) | 超线性收敛 |
| 有约束光滑 | 内点法 (IPM) | $O(\sqrt{n}\log(1/\epsilon))$ |
| 有约束光滑 | 投影梯度法 | $O(1/k)$ |
| 非光滑凸 | 次梯度法 | $O(1/\sqrt{k})$ |
| 复合优化 | 近端梯度法 (ISTA/FISTA) | $O(1/k)$ / $O(1/k^2)$ |
| 可分结构 | ADMM | 依赖于问题 |
| 大规模 | 随机梯度下降 (SGD) | $O(1/\sqrt{k})$ |

## 一、[[无约束凸优化算法]]

### 1.1 梯度下降法 (Gradient Descent)

**迭代**：$\mathbf{x}^{(k+1)} = \mathbf{x}^{(k)} - t_k \nabla f(\mathbf{x}^{(k)})$

**假设**：$f$ 凸，$\nabla f$ 为 $L$-Lipschitz连续

**步长选择**：
- **固定步长**：$t = 1/L$（保证收敛）
- **精确线搜索**：$t_k = \arg\min_t f(\mathbf{x}^{(k)} - t \nabla f^{(k)})$
- **回溯线搜索**（Armijo条件）：$f(\mathbf{x} - t\nabla f) \leq f(\mathbf{x}) - \alpha t \|\nabla f\|^2$

**收敛率**：
$$f(\mathbf{x}^{(k)}) - f(\mathbf{x}^*) \leq \frac{L\|\mathbf{x}^{(0)} - \mathbf{x}^*\|^2}{2k}$$

### 1.2 加速梯度下降 (Nesterov Accelerated Gradient)

**迭代**：
$$\begin{aligned} \mathbf{y}^{(k+1)} &= \mathbf{x}^{(k)} - t_k \nabla f(\mathbf{x}^{(k)}) \\ \mathbf{x}^{(k+1)} &= \mathbf{y}^{(k+1)} + \beta_k (\mathbf{y}^{(k+1)} - \mathbf{y}^{(k)}) \end{aligned}$$

其中 $\beta_k = \frac{k}{k+3}$（典型选择）。

**收敛率**：
$$f(\mathbf{x}^{(k)}) - f(\mathbf{x}^*) \leq \frac{2L\|\mathbf{x}^{(0)} - \mathbf{x}^*\|^2}{k^2}$$

> 对于一阶方法，$O(1/k^2)$ 是光滑凸优化的**下确界最优**速率！

### 1.3 Newton法

**迭代**：$\mathbf{x}^{(k+1)} = \mathbf{x}^{(k)} - t_k [\nabla^2 f(\mathbf{x}^{(k)})]^{-1} \nabla f(\mathbf{x}^{(k)})$

**Newton步径**：$\Delta \mathbf{x}_{nt} = -[\nabla^2 f]^{-1} \nabla f$

**Newton减量**：$\lambda(\mathbf{x}) = (\nabla f^T [\nabla^2 f]^{-1} \nabla f)^{1/2}$

**停止准则**：$\lambda(\mathbf{x})^2 / 2 \leq \epsilon$

**收敛性（三阶段）**：
1. **衰减阶段**：线性收敛
2. **二次收敛阶段**：$\|\mathbf{x}^{(k+1)} - \mathbf{x}^*\| \leq C\|\mathbf{x}^{(k)} - \mathbf{x}^*\|^2$
3. 仅需少量迭代（通常5-15次）达到高精度

**优劣**：
- 优点：二次收敛，极快
- 缺点：需要计算和存储Hessian，$O(n^3)$ 每步计算

### 1.4 拟Newton法 (BFGS / L-BFGS)

用梯度差分构造Hessian逆的近似 $\mathbf{H}_k \approx [\nabla^2 f]^{-1}$：

**BFGS更新**：
$$\mathbf{H}_{k+1} = \left(\mathbf{I} - \rho_k \mathbf{s}_k \mathbf{y}_k^T\right) \mathbf{H}_k \left(\mathbf{I} - \rho_k \mathbf{y}_k \mathbf{s}_k^T\right) + \rho_k \mathbf{s}_k \mathbf{s}_k^T$$

其中 $\mathbf{s}_k = \mathbf{x}^{(k+1)} - \mathbf{x}^{(k)}$, $\mathbf{y}_k = \nabla f^{(k+1)} - \nabla f^{(k)}$, $\rho_k = 1/(\mathbf{y}_k^T \mathbf{s}_k)$。

**L-BFGS**：仅存储最近 $m$ 对 $(\mathbf{s}_k, \mathbf{y}_k)$，存储量 $O(mn)$。

## 二、[[有约束凸优化算法]]

### 2.1 投影梯度法 (Projected Gradient Descent)

$$$\mathbf{x}^{(k+1)} = P_\mathcal{C}\left(\mathbf{x}^{(k)} - t_k \nabla f(\mathbf{x}^{(k)})\right)$$

其中 $P_\mathcal{C}$ 为向凸集 $\mathcal{C}$ 的**Euclid投影**。

- 要求投影计算简便（如非负卦限、范数球、半正定锥）
- 收敛率同梯度下降：$O(1/k)$

### 2.2 障碍法 (Barrier Method)

利用**对数障碍函数**将有约束问题转化为无约束问题序列：

$$\min_{\mathbf{x}} \quad t f_0(\mathbf{x}) + \phi(\mathbf{x})$$

其中 $\phi(\mathbf{x}) = -\sum_{i=1}^{m} \log(-f_i(\mathbf{x}))$ 是**对数障碍**。

**中心路径**：$\mathbf{x}^*(t)$ 对每个 $t > 0$ 的最优解轨迹，满足 $\lim_{t \to \infty} \mathbf{x}^*(t) = \mathbf{x}^*$。

**算法框架**：
1. 给定 $t^{(0)} > 0$, $\mu > 1$, 容差 $\epsilon$
2. **中心步**：从 $\mathbf{x}$ 开始，用Newton法计算 $\mathbf{x}^*(t)$
3. **更新**：$\mathbf{x} \leftarrow \mathbf{x}^*(t)$
4. **停止**：若 $m/t < \epsilon$，返回 $\mathbf{x}$
5. **增大**：$t \leftarrow \mu t$，返回第2步

**复杂度**：$O(\sqrt{m}\log(1/\epsilon))$ 次Newton迭代。

### 2.3 原始-对偶内点法 (Primal-Dual IPM)

直接求解修改后的KKT条件（扰动KKT）：

$$\begin{aligned} \nabla f_0(\mathbf{x}) + \sum \lambda_i \nabla f_i(\mathbf{x}) + \mathbf{A}^T \boldsymbol{\nu} &= 0 \\ -\lambda_i f_i(\mathbf{x}) &= 1/t \\ f_i(\mathbf{x}) &\leq 0 \\ \lambda_i &\geq 0 \end{aligned}$$

用Newton法一步迭代，同时更新原始和对偶变量。这是**实际应用中最高效**的凸优化求解方法。

## 三、[[次梯度方法]] (Subgradient Method)

用于**非光滑凸优化**，如 $\ell_1$ 范数、hinge loss等。

### 3.1 次梯度

向量 $\mathbf{g}$ 称为 $f$ 在 $\mathbf{x}$ 处的**次梯度**，若：

$$f(\mathbf{y}) \geq f(\mathbf{x}) + \mathbf{g}^T (\mathbf{y} - \mathbf{x}), \quad \forall \mathbf{y}$$

所有次梯度的集合记为 $\partial f(\mathbf{x})$（**次微分**）。

### 3.2 次梯度法

$$\mathbf{x}^{(k+1)} = \mathbf{x}^{(k)} - t_k \mathbf{g}^{(k)}, \quad \mathbf{g}^{(k)} \in \partial f(\mathbf{x}^{(k)})$$

**关键差异**：
- **不是下降方向**：$\mathbf{g}^{(k)}$ 不保证 $f$ 下降
- 步长 $t_k$ 必须**事先确定**：$t_k \to 0$, $\sum t_k = \infty$
- 需要保留**历史最好解**：$f_{\text{best}}^{(k)} = \min(f_{\text{best}}^{(k-1)}, f(\mathbf{x}^{(k)}))$

**收敛率**：
$$f_{\text{best}}^{(k)} - f^* \leq \frac{G\|\mathbf{x}^{(0)} - \mathbf{x}^*\|}{\sqrt{k}}$$

其中 $G$ 为次梯度的上界。

## 四、[[近端梯度法与ADMM]]

### 4.1 近端算子 (Proximal Operator)

函数 $h$ 的近端算子：

$$\text{prox}_{t h}(\mathbf{v}) = \arg\min_{\mathbf{x}} \left(h(\mathbf{x}) + \frac{1}{2t}\|\mathbf{x} - \mathbf{v}\|_2^2\right)$$

常见近端算子的闭式解：
- $h(\mathbf{x}) = \|\mathbf{x}\|_1$ ⇒ 软阈值（soft-thresholding）
- $h(\mathbf{x}) = I_\mathcal{C}(\mathbf{x})$ ⇒ Euclid投影
- $h(\mathbf{x}) = \|\mathbf{x}\|_2$ ⇒ 群软阈值

### 4.2 近端梯度法 (Proximal Gradient / ISTA)

用于复合目标 $f(\mathbf{x}) = g(\mathbf{x}) + h(\mathbf{x})$，其中 $g$ 光滑凸，$h$ 非光滑凸（但近端算子计算简便）。

$$\mathbf{x}^{(k+1)} = \text{prox}_{t_k h}\left(\mathbf{x}^{(k)} - t_k \nabla g(\mathbf{x}^{(k)})\right)$$

**加速版（FISTA）**：结合Nesterov动量，收敛率从 $O(1/k)$ 提升至 $O(1/k^2)$。

### 4.3 ADMM (Alternating Direction Method of Multipliers)

用于带线性约束的可分优化：

$$\begin{aligned} \min_{\mathbf{x}, \mathbf{z}} \quad & f(\mathbf{x}) + g(\mathbf{z}) \\ \text{s.t.} \quad & \mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{z} = \mathbf{c} \end{aligned}$$

**增广Lagrange函数**：
$$L_\rho(\mathbf{x}, \mathbf{z}, \mathbf{y}) = f(\mathbf{x}) + g(\mathbf{z}) + \mathbf{y}^T (\mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{z} - \mathbf{c}) + \frac{\rho}{2}\|\mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{z} - \mathbf{c}\|_2^2$$

**ADMM迭代**：
$$\begin{aligned} \mathbf{x}^{(k+1)} &= \arg\min_{\mathbf{x}} L_\rho(\mathbf{x}, \mathbf{z}^{(k)}, \mathbf{y}^{(k)}) \\ \mathbf{z}^{(k+1)} &= \arg\min_{\mathbf{z}} L_\rho(\mathbf{x}^{(k+1)}, \mathbf{z}, \mathbf{y}^{(k)}) \\ \mathbf{y}^{(k+1)} &= \mathbf{y}^{(k)} + \rho (\mathbf{A}\mathbf{x}^{(k+1)} + \mathbf{B}\mathbf{z}^{(k+1)} - \mathbf{c}) \end{aligned}$$

**特点**：
- 变量分块，每步子问题可并行
- 可处理分布式/大规模问题
- 对凸问题收敛保证，但速度依赖于 $\rho$ 的选择

## 五、[[随机梯度下降 SGD]]

### 5.1 基本SGD

用于 $\min_\mathbf{x} \frac{1}{N}\sum_{i=1}^{N} f_i(\mathbf{x})$（有限和结构）：

$$\mathbf{x}^{(k+1)} = \mathbf{x}^{(k)} - t_k \nabla f_{i_k}(\mathbf{x}^{(k)})$$

其中 $i_k$ 从 $\{1,\ldots,N\}$ 随机/循环选取。

**收敛条件**：$\sum t_k = \infty$, $\sum t_k^2 < \infty$（典型：$t_k \propto 1/\sqrt{k}$）

**收敛率**：$O(1/\sqrt{k})$（次线性收敛）

### 5.2 方差缩减方法

- **SAG**（Stochastic Average Gradient）：维护所有梯度估计的滑动平均
- **SVRG**（Stochastic Variance Reduced Gradient）：周期性计算全梯度，修正随机梯度
- **SAGA**：SAG的改进版本

这些方法可恢复梯度下降的 $O(1/k)$ 线性收敛率。

### 5.3 自适应方法

- **AdaGrad**：基于历史梯度平方和学习率
- **RMSProp**：指数移动平均调整学习率
- **Adam**：结合动量与自适应学习率（实践中非常常用）

## 七、收敛速度对比速查表

| 方法 | 迭代复杂度 | 每步计算量 | 适用问题 |
|------|-----------|-----------|---------|
| GD | $O(1/\epsilon)$ | $O(n)$ | 光滑无约束 |
| Nesterov AGD | $O(1/\sqrt{\epsilon})$ | $O(n)$ | 光滑无约束 |
| Newton | $O(\log\log(1/\epsilon))$ | $O(n^3)$ | 小规模无约束 |
| L-BFGS | 超线性 | $O(mn)$ | 中大规模无约束 |
| 投影梯度 | $O(1/\epsilon)$ | $O(n)$ + 投影 | 有约束 |
| 内点法 | $O(\sqrt{m}\log(1/\epsilon))$ | $O(n^3)$/Newton步 | 有约束精解 |
| 次梯度法 | $O(1/\epsilon^2)$ | $O(n)$ | 非光滑 |
| FISTA | $O(1/\sqrt{\epsilon})$ | $O(n)$ | 复合优化 |
| SGD | $O(1/\epsilon^2)$ | $O(n)$ | 大规模/在线 |
