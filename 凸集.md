# 凸集 (Convex Sets)

凸集是凸优化的几何基础。一个优化问题的**可行域**为凸集，是保证问题为凸优化问题的必要条件之一。

## 一、基本定义

### 1.1 仿射集 (Affine Set)

集合 $C \subseteq \mathbb{R}^n$ 称为**仿射集**，若对任意 $\mathbf{x}_1, \mathbf{x}_2 \in C$ 和任意 $\theta \in \mathbb{R}$，有：

$$\theta \mathbf{x}_1 + (1-\theta)\mathbf{x}_2 \in C$$

即：通过 $C$ 中任意两点的**整条直线**仍在 $C$ 中。

### 1.2 凸集 (Convex Set)

集合 $C \subseteq \mathbb{R}^n$ 称为**凸集**，若对任意 $\mathbf{x}_1, \mathbf{x}_2 \in C$ 和任意 $\theta \in [0,1]$，有：

$$\theta \mathbf{x}_1 + (1-\theta)\mathbf{x}_2 \in C$$

即：连接 $C$ 中任意两点的**线段**完全包含在 $C$ 中。

> **直观理解**：凸集 = "没有凹陷"的集合。仿射集 ⊂ 凸集（仿射集要求整条直线，凸集只要求线段）。

### 1.3 凸锥 (Convex Cone)

集合 $C$ 称为**锥**，若对任意 $\mathbf{x} \in C$ 和 $\theta \geq 0$，有 $\theta \mathbf{x} \in C$。

集合 $C$ 称为**凸锥**，若对任意 $\mathbf{x}_1, \mathbf{x}_2 \in C$ 和任意 $\theta_1, \theta_2 \geq 0$，有：

$$\theta_1 \mathbf{x}_1 + \theta_2 \mathbf{x}_2 \in C$$

即：凸锥 = 既是凸集又是锥。

> **包含关系**：仿射集 ⊂ 凸锥（非负组合）⊂ 凸集

## 二、典型凸集

| 集合类型 | 定义 | 凸性 |
|---------|------|------|
| 超平面 | $\{\mathbf{x} \mid \mathbf{a}^T \mathbf{x} = b\}$ | 仿射集 |
| 半空间 | $\{\mathbf{x} \mid \mathbf{a}^T \mathbf{x} \leq b\}$ | 凸集 |
| Euclid球 | $\{\mathbf{x} \mid \|\mathbf{x}-\mathbf{x}_c\|_2 \leq r\}$ | 凸集 |
| 椭球 | $\{\mathbf{x} \mid (\mathbf{x}-\mathbf{x}_c)^T \mathbf{P}^{-1}(\mathbf{x}-\mathbf{x}_c) \leq 1\}$, $\mathbf{P} \succ 0$ | 凸集 |
| 多面体 | $\{\mathbf{x} \mid \mathbf{A}\mathbf{x} \preceq \mathbf{b}, \mathbf{C}\mathbf{x} = \mathbf{d}\}$ | 凸集 |
| 范数球 | $\{\mathbf{x} \mid \|\mathbf{x}-\mathbf{x}_c\| \leq r\}$ | 凸集 |
| 范数锥 | $\{(\mathbf{x}, t) \mid \|\mathbf{x}\| \leq t\}$ | 凸锥 |
| 半正定锥 | $\mathbf{S}_+^n = \{\mathbf{X} \in \mathbf{S}^n \mid \mathbf{X} \succeq 0\}$ | 凸锥 |

## 三、[[保凸运算]]

下列运算**保持凸性**：

1. **任意交**：$\bigcap_{\alpha} C_\alpha$ 是凸集
2. **仿射函数**：$f(\mathbf{x}) = \mathbf{A}\mathbf{x} + \mathbf{b}$ 下，$f(C)$ 和 $f^{-1}(D)$ 均为凸
3. **透视函数**：$P(\mathbf{x}, t) = \mathbf{x}/t$, $\text{dom }P = \mathbb{R}^n \times \mathbb{R}_{++}$
4. **线性分式函数**：$f(\mathbf{x}) = (\mathbf{A}\mathbf{x}+\mathbf{b})/(\mathbf{c}^T\mathbf{x}+d)$

> 更多细节见 [[保凸运算]]

## 四、[[分离超平面定理]]

分离超平面定理是凸分析中**最重要的几何定理**。

**定理**：设 $C, D \subseteq \mathbb{R}^n$ 为两个不相交的凸集（$C \cap D = \varnothing$），则存在 $\mathbf{a} \neq \mathbf{0}$ 和 $b \in \mathbb{R}$，使得：

$$\mathbf{a}^T \mathbf{x} \leq b, \quad \forall \mathbf{x} \in C$$
$$\mathbf{a}^T \mathbf{x} \geq b, \quad \forall \mathbf{x} \in D$$

超平面 $\{\mathbf{x} \mid \mathbf{a}^T \mathbf{x} = b\}$ 将 $C$ 与 $D$ **分离**。

**严格分离**：若 $C$ 为闭凸集，$D$ 为紧集且 $C \cap D = \varnothing$，则存在超平面**严格分离**两者。

## 五、[[支撑超平面]]

给定集合 $C \subseteq \mathbb{R}^n$ 及其边界点 $\mathbf{x}_0 \in \text{bd }C$，若：

$$\mathbf{a}^T \mathbf{x} \leq \mathbf{a}^T \mathbf{x}_0, \quad \forall \mathbf{x} \in C$$

则超平面 $\{\mathbf{x} \mid \mathbf{a}^T \mathbf{x} = \mathbf{a}^T \mathbf{x}_0\}$ 称为 $C$ 在 $\mathbf{x}_0$ 处的**支撑超平面**。

**支撑超平面定理**：对任何非空凸集 $C$ 及其边界上的任一点，存在该点的支撑超平面。

> 支撑超平面定理 ⇒ 分离超平面定理

## 六、[[对偶锥与广义不等式]]

### 6.1 对偶锥

锥 $K$ 的**对偶锥**定义为：

$$K^* = \{\mathbf{y} \mid \mathbf{y}^T \mathbf{x} \geq 0, \forall \mathbf{x} \in K\}$$

- $K^*$ 始终为闭凸锥
- 若 $K$ 为真锥（闭、尖、实心），则 $K^*$ 亦为真锥
- 对偶锥的对偶锥：$K^{**} = K$（当 $K$ 为闭凸锥时）

### 6.2 广义不等式

由真锥 $K$ 引入**偏序**：

$$\mathbf{x} \preceq_K \mathbf{y} \iff \mathbf{y} - \mathbf{x} \in K$$
$$\mathbf{x} \prec_K \mathbf{y} \iff \mathbf{y} - \mathbf{x} \in \text{int }K$$

广义不等式保持了**加法保序**、**传递性**和**非负乘子保序**等性质。

## 七、凸集的拓扑性质

- **凸包** $\text{conv }S$：包含 $S$ 的最小凸集
- **仿射包** $\text{aff }S$：包含 $S$ 的最小仿射集
- **锥包** $\text{cone }S$：包含 $S$ 的最小凸锥
- **相对内部** $\text{relint }C$：相对于 $\text{aff }C$ 的内部

**重要事实**：
- 凸集的闭包仍是凸集
- 凸集的相对内部非空（若 $C \neq \varnothing$）
- Carathéodory定理：$\mathbb{R}^n$ 中任何凸组合可由至多 $n+1$ 个点的凸组合表示
