# 凸优化 (Convex Optimization)

凸优化是数值分析与数学优化的核心分支，研究**凸函数**在**凸集**上的最小化问题。其核心优势在于：任何局部最优解必然是全局最优解。

> **入门首选**：[[凸优化导学]] — 一条线索从动机到算法，串起全部知识点。

## 核心定理链

```
凸集定义 → 保凸运算 → 分离超平面定理
                            ↓
凸函数定义 → 一阶/二阶条件 → Jensen不等式
                            ↓
        凸优化问题标准形式
                            ↓
    Lagrange对偶 → 弱对偶 → 强对偶 (Slater条件)
                            ↓
                      KKT条件
                            ↓
                    数值算法求解
```

---

## 为什么要研究凸优化？

1. **全局最优性保证**：局部最优 = 全局最优，没有伪局部极小值
2. **可计算性**：存在多项式时间算法（内点法）
3. **对偶理论完备**：KKT条件提供了最优性的充要条件
4. **应用广泛**：控制、信号处理、机器学习、金融、通信等
5. **作为非凸问题的基石**：很多非凸优化方法（如凸松弛）依赖于凸优化
