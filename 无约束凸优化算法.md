# 无约束凸优化算法

无约束凸优化是[[凸优化算法]]的起点。即使是有约束问题（通过[[有约束凸优化算法|内点法]]），最终也转化为一系列无约束子问题。

[[凸优化算法]] | [[凸函数一阶条件]] | [[凸函数二阶条件]]

---

## 一、问题形式

$$\min_{\mathbf{x} \in \mathbb{R}^n} f(\mathbf{x})$$

假设：$f$ 凸、可微（或二阶可微），最优值 $p^* = f(\mathbf{x}^*) > -\infty$ 且最优解 $\mathbf{x}^*$ 存在。

由[[凸函数一阶条件]]：$\nabla f(\mathbf{x}^*) = \mathbf{0}$ 是充要最优性条件。

---

## 二、梯度下降法 (Gradient Descent)

### 2.1 迭代

$$\mathbf{x}^{(k+1)} = \mathbf{x}^{(k)} - t_k \nabla f(\mathbf{x}^{(k)})$$

### 2.2 假设

$\nabla f$ 为 $L$-Lipschitz 连续（称为 $L$-光滑）：

$$\|\nabla f(\mathbf{x}) - \nabla f(\mathbf{y})\|_2 \leq L\|\mathbf{x} - \mathbf{y}\|_2$$

### 2.3 步长策略

**固定步长**：$t = 1/L$

保证：
$$f(\mathbf{x}^{(k+1)}) \leq f(\mathbf{x}^{(k)}) - \frac{1}{2L}\|\nabla f(\mathbf{x}^{(k)})\|_2^2$$

**回溯线搜索（Backtracking Line Search / Armijo）**：

给定参数 $\alpha \in (0, 0.5)$, $\beta \in (0,1)$。每步从 $t = 1$ 开始，当：

$$f(\mathbf{x} - t\nabla f) > f(\mathbf{x}) - \alpha t \|\nabla f\|_2^2$$

时令 $t \leftarrow \beta t$。保证充分下降。

**精确线搜索**：$t_k = \arg\min_{t > 0} f(\mathbf{x}^{(k)} - t \nabla f^{(k)})$（很少实际使用）。

### 2.4 收敛率

$$f(\mathbf{x}^{(k)}) - p^* \leq \frac{L\|\mathbf{x}^{(0)} - \mathbf{x}^*\|_2^2}{2k} = O\left(\frac{1}{k}\right)$$

**强凸情形**（$\mu$-强凸 + $L$-光滑）：

$$f(\mathbf{x}^{(k)}) - p^* \leq \frac{L}{2}\left(1 - \frac{\mu}{L}\right)^k \|\mathbf{x}^{(0)} - \mathbf{x}^*\|_2^2 = O\left(\left(1 - \frac{1}{\kappa}\right)^k\right)$$

其中 $\kappa = L/\mu$ 为条件数。这是**线性收敛**。

### 2.5 梯度下降的收敛行为

- 条件数 $\kappa \approx 1$：快速收敛（接近球形等高线）
- 条件数 $\kappa \gg 1$：慢速收敛（锯齿路径，沿"峡谷"来回震荡）

## 三、Nesterov加速梯度下降 (Accelerated Gradient Descent, AGD)

### 3.1 迭代

$$\begin{aligned} \mathbf{y}^{(k)} &= \mathbf{x}^{(k)} + \beta_k (\mathbf{x}^{(k)} - \mathbf{x}^{(k-1)}) \quad \text{(外推/动量)} \\ \mathbf{x}^{(k+1)} &= \mathbf{y}^{(k)} - t_k \nabla f(\mathbf{y}^{(k)}) \end{aligned}$$

其中 $\beta_k = \frac{k-1}{k+2}$ 或 $\beta_k = \frac{\theta_k(1-\theta_k)}{\theta_k^2 + \theta_{k+1}}$ 是精心设计的动量系数。

### 3.2 收敛率

**光滑凸**：$f(\mathbf{x}^{(k)}) - p^* \leq \frac{2L\|\mathbf{x}^{(0)} - \mathbf{x}^*\|_2^2}{k^2} = O\left(\frac{1}{k^2}\right)$

**光滑强凸**：$f(\mathbf{x}^{(k)}) - p^* \leq O\left(\left(1 - \frac{1}{\sqrt{\kappa}}\right)^k\right)$

### 3.3 最优性

对于仅用一阶信息的光滑凸优化，$O(1/k^2)$ 是**下确界**（理论上最优）。Nesterov方法达到了这个下界。

**动量机制的直觉**：前一步的动量"推动"迭代沿惯性方向前进，避免了锯齿路径。

## 四、Newton法

### 4.1 迭代

$$\mathbf{x}^{(k+1)} = \mathbf{x}^{(k)} - t_k [\nabla^2 f(\mathbf{x}^{(k)})]^{-1} \nabla f(\mathbf{x}^{(k)})$$

### 4.2 Newton步径

$$\Delta \mathbf{x}_{nt} = -[\nabla^2 f]^{-1} \nabla f$$

$\Delta \mathbf{x}_{nt}$ 来自对 $f$ 在当前点的**二阶Taylor近似**求最小：

$$\min_{\mathbf{v}} f(\mathbf{x}) + \nabla f(\mathbf{x})^T \mathbf{v} + \frac{1}{2}\mathbf{v}^T \nabla^2 f(\mathbf{x}) \mathbf{v}$$

### 4.3 Newton减量

$$\lambda(\mathbf{x}) = \left(\nabla f(\mathbf{x})^T [\nabla^2 f(\mathbf{x})]^{-1} \nabla f(\mathbf{x})\right)^{1/2}$$

- 这是以Hessian度量的梯度长度
- **仿射不变性**：$\lambda(\mathbf{x})$ 在线性变换下不变（梯度下降不具备此性质）

### 4.4 停止准则

$$\frac{\lambda(\mathbf{x})^2}{2} \leq \epsilon$$

因为 $f(\mathbf{x}) - p^* \leq \frac{\lambda(\mathbf{x})^2}{2}$（对自和谐函数）。

### 4.5 收敛性（三阶段）

**阶段1（衰减阶段）**：线性收敛
**阶段2（二次收敛阶段）**：一旦进入，$\|\mathbf{x}^{(k+1)} - \mathbf{x}^*\| \leq C\|\mathbf{x}^{(k)} - \mathbf{x}^*\|^2$

实际中，Newton法通常**5-15步**即收敛到高精度。

### 4.6 复杂度

- 每步：$O(n^3)$（求Hessian逆）+ $O(n^2)$（Hessian计算）
- 对 $n$ 很大时不实用

## 五、拟Newton法 (BFGS / L-BFGS)

### 5.1 核心思想

用梯度差分**近似Hessian的逆** $\mathbf{H}_k \approx [\nabla^2 f(\mathbf{x}^{(k)})]^{-1}$，而无需显式计算Hessian。

### 5.2 BFGS更新公式

$$\mathbf{H}_{k+1} = (\mathbf{I} - \rho_k \mathbf{s}_k \mathbf{y}_k^T) \mathbf{H}_k (\mathbf{I} - \rho_k \mathbf{y}_k \mathbf{s}_k^T) + \rho_k \mathbf{s}_k \mathbf{s}_k^T$$

其中：
- $\mathbf{s}_k = \mathbf{x}^{(k+1)} - \mathbf{x}^{(k)}$
- $\mathbf{y}_k = \nabla f^{(k+1)} - \nabla f^{(k)}$
- $\rho_k = 1/(\mathbf{y}_k^T \mathbf{s}_k)$

**性质**：
- $\mathbf{H}_{k+1} \succ 0$（保持正定性）
- 满足**割线方程** $\mathbf{H}_{k+1}\mathbf{y}_k = \mathbf{s}_k$

### 5.3 L-BFGS（Limited Memory BFGS）

仅存储最近 $m$ 对 $(\mathbf{s}_k, \mathbf{y}_k)$，隐式计算 $\mathbf{H}_k \mathbf{g}$：
- 存储需求：$O(mn)$ 而非 $O(n^2)$
- 适合大规模问题（$n$ 可达百万）
- 实践中 $m = 5 \sim 20$ 即可

### 5.4 收敛率

超线性收敛（介于线性与二次之间），实践中接近Newton法。

## 六、共轭梯度法 (Conjugate Gradient)

### 6.1 适用场景

$$\min \frac{1}{2}\mathbf{x}^T \mathbf{P}\mathbf{x} - \mathbf{q}^T\mathbf{x}$$

或对一般凸函数做线性化近似。

### 6.2 算法

生成方向 $\mathbf{p}^{(k)}$（$\mathbf{P}$-共轭），沿此方向精确搜索。

**优点**：
- 无需存储Hessian（只需矩阵-向量积）
- 对二次型，$n$ 步内精确收敛
- 对一般函数，是Newton和梯度下降的折中

---

## 七、算法选择指南

| 规模 | 精度需求 | 推荐方法 |
|------|---------|---------|
| 小 ($n<1000$) | 高精度 | Newton法 |
| 中 ($1000<n<10^5$) | 中等 | L-BFGS |
| 大 ($n>10^5$) | 中等 | 加速梯度下降 (Nesterov) |
| 极大 ($n>10^7$) | 低至中 | SGD / Adam（见[[随机梯度下降 SGD]]） |
| 二次问题 | 精确 | 共轭梯度法 |
| 非光滑 | — | [[次梯度方法]] |
