# 保凸运算

哪些运算将一个（或多个）[[凸集]]映射为另一个凸集？掌握这些运算可以高效构造复杂凸可行域。

[[凸集]] | [[凸集定义与性质]] | [[凸函数]] | [[保凸运算与复合规则]]

---

## 1. 任意交 (Intersection)

若 $C_\alpha$ 对每个 $\alpha \in \mathcal{A}$ 是凸集，则：

$$\bigcap_{\alpha \in \mathcal{A}} C_\alpha \text{ 是凸集}$$

**意义**：不等式约束组 $f_i(\mathbf{x}) \leq 0$ 定义的可行域 = 无穷多个半空间的交 = 凸集。

**推论**：任意不等式组只要每个子水平集是凸的，可行域即凸。

---

## 2. 仿射函数 (Affine Function)

设 $f: \mathbb{R}^n \to \mathbb{R}^m$ 为仿射函数：$f(\mathbf{x}) = \mathbf{A}\mathbf{x} + \mathbf{b}$（$\mathbf{A} \in \mathbb{R}^{m \times n}$, $\mathbf{b} \in \mathbb{R}^m$）。

### 2.1 像 (Image)

若 $C \subseteq \mathbb{R}^n$ 为凸集，则：

$$f(C) = \{\mathbf{A}\mathbf{x} + \mathbf{b} \mid \mathbf{x} \in C\} \subseteq \mathbb{R}^m \text{ 是凸集}$$

### 2.2 原像 (Inverse Image)

若 $D \subseteq \mathbb{R}^m$ 为凸集，则：

$$f^{-1}(D) = \{\mathbf{x} \in \mathbb{R}^n \mid \mathbf{A}\mathbf{x} + \mathbf{b} \in D\} \text{ 是凸集}$$

### 2.3 典型应用

- **缩放与平移**：$aC + b$（$a \in \mathbb{R}$, $b \in \mathbb{R}^n$）保持凸性
- **Cartesian积**：$C_1 \times C_2$ 是凸集（取 $f$ 为投影的逆）
- **和**：$C_1 + C_2 = \{\mathbf{x}_1 + \mathbf{x}_2 \mid \mathbf{x}_1 \in C_1, \mathbf{x}_2 \in C_2\}$ 是凸集
- **部分和**：$C_1 \times C_2$ 在线性映射 $(x_1,x_2) \mapsto x_1 + x_2$ 下的像

---

## 3. 透视函数 (Perspective Function)

$P: \mathbb{R}^{n+1} \to \mathbb{R}^n$（定义域 $\mathbb{R}^n \times \mathbb{R}_{++}$）：

$$P(\mathbf{z}, t) = \frac{\mathbf{z}}{t}, \quad \mathbf{z} \in \mathbb{R}^n, t > 0$$

- 若 $C \subseteq \text{dom }P$ 是凸集，则 $P(C)$ 是凸集
- 若 $D \subseteq \mathbb{R}^n$ 是凸集，则 $P^{-1}(D)$ 是凸集

**物理意义**：透视函数将 $\mathbb{R}^{n+1}$ 中的点（齐次坐标）投射到 $\mathbb{R}^n$（针孔相机模型）。

---

## 4. 线性分式函数 (Linear-Fractional Function)

$f: \mathbb{R}^n \to \mathbb{R}^m$：

$$f(\mathbf{x}) = \frac{\mathbf{A}\mathbf{x} + \mathbf{b}}{\mathbf{c}^T \mathbf{x} + d}, \quad \text{dom }f = \{\mathbf{x} \mid \mathbf{c}^T \mathbf{x} + d > 0\}$$

### 保凸性质

若 $C \subseteq \text{dom }f$ 为凸集，则 $f(C)$ 是凸集。

### 与透视函数的关系

线性分式函数 = 透视函数 + 仿射函数的复合：

$$f(\mathbf{x}) = P(\mathbf{A}\mathbf{x} + \mathbf{b}, \mathbf{c}^T\mathbf{x} + d)$$

**应用**：条件概率、Bayes更新后验参数保持凸性。

---

## 5. Minkowski和与直和

### Minkowski和

$$C + D = \{\mathbf{x} + \mathbf{y} \mid \mathbf{x} \in C, \mathbf{y} \in D\}$$

若 $C, D$ 凸，则 $C + D$ 凸。

### 直和 (Cartesian积)

$$C \times D = \{(\mathbf{x}, \mathbf{y}) \mid \mathbf{x} \in C, \mathbf{y} \in D\}$$

凸集的Cartesian积仍凸。

---

## 6. 凸包与闭凸包

- $\text{conv}(C \cup D)$ 是凸集（由构造即凸）
- $\text{cl }C$（闭包）若 $C$ 凸，则 $\text{cl }C$ 凸

---

## 运算保凸性速查表

| 运算 | 输入 | 输出 | 保凸？ |
|------|------|------|-------|
| 交 | $\{C_\alpha\}$ 凸 | $\bigcap C_\alpha$ | ✓ |
| 仿射像 | $C$ 凸 | $\mathbf{A}C + \mathbf{b}$ | ✓ |
| 仿射原像 | $D$ 凸 | $\mathbf{A}^{-1}(D)$ | ✓ |
| 透视像 | $C$ 凸 | $P(C)$ | ✓ |
| 透视原像 | $D$ 凸 | $P^{-1}(D)$ | ✓ |
| 线性分式像 | $C$ 凸 | $f(C)$ | ✓ |
| Minkowski和 | $C,D$ 凸 | $C + D$ | ✓ |
| Cartesian积 | $C,D$ 凸 | $C \times D$ | ✓ |
| 凸包 | 任意集合 | $\text{conv }S$ | ✓（定义） |
| 并 | $C,D$ 凸 | $C \cup D$ | ✗（一般否） |
| 补 | $C$ 凸 | $\mathbb{R}^n \setminus C$ | ✗（一般否） |
