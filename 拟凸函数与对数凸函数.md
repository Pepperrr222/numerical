# 拟凸函数与对数凸函数

并非所有有用的函数都是凸函数。**拟凸函数**和**对数凸函数**是两种重要的广义凸性，它们保留了凸函数的部分但非全部优良性质。

## 一、拟凸函数 (Quasiconvex Functions)

### 1.1 定义

函数 $f: \mathbb{R}^n \to \mathbb{R}$ 称为**拟凸函数**，若 $\text{dom }f$ 是凸集，且**所有下水平集**为凸：

$$S_\alpha = \{\mathbf{x} \in \text{dom }f \mid f(\mathbf{x}) \leq \alpha\} \text{ 为凸集}, \quad \forall \alpha \in \mathbb{R}$$

### 1.2 等价定义

$$f(\theta\mathbf{x} + (1-\theta)\mathbf{y}) \leq \max\{f(\mathbf{x}), f(\mathbf{y})\}, \quad \forall \mathbf{x}, \mathbf{y} \in \text{dom }f, \theta \in [0,1]$$

**拟凹**：$f$ 拟凸 ⇔ $-f$ 拟凹。或所有上水平集 $\{\mathbf{x} \mid f(\mathbf{x}) \geq \alpha\}$ 是凸集。

### 1.3 凸 vs 拟凸

| 性质 | 凸函数 | 拟凸函数 |
|------|--------|---------|
| 定义 | $f(\theta x+(1-\theta)y) \leq \theta f(x)+(1-\theta)f(y)$ | $f(\theta x+(1-\theta)y) \leq \max\{f(x),f(y)\}$ |
| 下水平集 | 凸 | 凸 |
| 非负加权和 | 保持 | **不一定保持** |
| 局部最小 ⇒ 全局最小 | 总是 | **不一定**（可能有"平坦"区域） |
| Jensen不等式 | $f(\mathbb{E}[X]) \leq \mathbb{E}[f(X)]$ | 不成立 |

## 二、拟凸函数的判定

### 2.1 一阶条件（可微拟凸）

$f$ 可微。$f$ 为拟凸 **当且仅当**：

$$f(\mathbf{y}) \leq f(\mathbf{x}) \;\Rightarrow\; \nabla f(\mathbf{x})^T (\mathbf{y} - \mathbf{x}) \leq 0$$

**几何意义**：梯度 $\nabla f(\mathbf{x})$ 定义了一个分离超平面，将 $\mathbf{x}$ 所在下水平集与函数的更低值区域分开。

### 2.2 二阶条件（一维）

$f: \mathbb{R} \to \mathbb{R}$，$f'(x) = 0 \Rightarrow f''(x) > 0$：在驻点处严格凸 ⇒ 拟凸。

**注意**：对 $n > 1$，没有简单的"拟凸 ⇔ Hessian某种正定性"的二阶条件。

### 2.3 复合表示

$$f \text{ 拟凸} \iff f(\mathbf{x}) = \phi(g(\mathbf{x}))$$

其中 $g$ 是凸函数，$\phi$ 是 $\mathbb{R}$ 上的**非减**函数。

（这个表示并不总是存在，但有此形式的函数必为拟凸）

## 三、常见的拟凸函数

### 3.1 比值函数

$$f(\mathbf{x}) = \frac{\mathbf{a}^T\mathbf{x} + b}{\mathbf{c}^T\mathbf{x} + d}, \quad \text{dom }f = \{\mathbf{x} \mid \mathbf{c}^T\mathbf{x} + d > 0\}$$

在 $\text{dom }f$ 上既是拟凸也是拟凹（**拟线性**）。

### 3.2 距离比

$$f(\mathbf{x}) = \frac{\|\mathbf{x} - \mathbf{a}\|_2}{\|\mathbf{x} - \mathbf{b}\|_2}$$

（广义Apollonius问题）

### 3.3 内部收益率（IRR）

IRR是现金流的拟凸函数，尽管不是凸函数。

### 3.4 误码率（通信）

某些信噪比条件下，误码率是发射功率的拟凸函数。

## 四、拟凸优化

### 4.1 一般拟凸优化

$$\begin{aligned} \min \quad & f_0(\mathbf{x}) \\ \text{s.t.} \quad & f_i(\mathbf{x}) \leq 0 \\ & \mathbf{A}\mathbf{x} = \mathbf{b} \end{aligned}$$

其中 $f_0$ 拟凸，$f_i$ 凸。

可通过求解一族**凸可行性问题**来求解拟凸优化（**二分法**）：

对给定的 $t$，检查是否存在 $\mathbf{x}$ 满足 $f_0(\mathbf{x}) \leq t$ 且满足所有约束。用二分法搜索最小的可行 $t$。

### 4.2 二分法框架

```
给定: 区间 [l, u] 包含 p*, 容差 ε
while u - l > ε:
    t = (l + u) / 2
    求解凸可行性问题: 是否存在 x 满足
        f₀(x) ≤ t, fᵢ(x) ≤ 0, Ax = b?
    if 可行: u = t
    else: l = t
return u
```

每次迭代求解一个**凸可行性问题**，可用[[有约束凸优化算法|内点法]]处理。

## 五、对数凸函数 (Log-Convex Functions)

### 5.1 定义

函数 $f: \mathbb{R}^n \to \mathbb{R}_{++}$（正值）称为**对数凸函数**，若 $\log f$ 为凸函数。

等价地：

$$f(\theta\mathbf{x} + (1-\theta)\mathbf{y}) \leq f(\mathbf{x})^\theta \cdot f(\mathbf{y})^{1-\theta}$$

### 5.2 基本性质

- **凸 ⇒ 对数凸不成立**（方向错误：凸函数的 log 不一定是凸的）
- **对数凸 ⇒ 凸**（$\log f$ 凸 ⇔ $f$ 的值取 $\exp$ 后是凸的）**不成立**（反例：$f(x)=x$ 是对数凹的，$f(x)=e^{-x^2}$ 是对数凹的）
- **对数凸函数本身必为凸函数**（由于 $\exp$ 是凸非减函数，复合规则给出 $f = \exp(\log f)$ 是凸的）

> 包含关系：对数凸 ⊂ 凸 ⊂ 拟凸

### 5.3 常见对数凸函数

- $\Gamma$ 函数（Gamma函数）：$\Gamma(x)$ 在 $x > 0$ 上对数凸
- 正态分布的累积分布函数：$\Phi(x)$ 是对数凹的
- 多项分布的概率质量函数（参数固定，变量为概率向量时）

---

## 六、对数凹函数 (Log-Concave Functions)

### 6.1 定义

$f > 0$ 且 $\log f$ 为凹函数。

等价：$f = e^{\phi}$，$\phi$ 凹。

### 6.2 重要性质

- 对数凹函数的**卷积**仍为对数凹函数（Prékopa-Leindler不等式）
- 正态分布、指数分布、均匀分布的**概率密度函数**均为对数凹函数
- 对数凹分布具有良好的集中性和等周不等式

### 6.3 典型对数凹函数

- 高斯PDF：$f(x) = e^{-x^2/2}$ ⇒ $\log f(x) = -x^2/2$（凹）
- 指数分布PDF：$f(x) = \lambda e^{-\lambda x}$ ⇒ $\log f(x) = \log\lambda - \lambda x$（凹）
- Laplace分布PDF：$f(x) \propto e^{-|x|}$ ⇒ $\log f(x) = -|x|$（凹）

## 七、凸 → 对数凸 → 拟凸 的关系图

```
                ┌──────────────────────────────┐
                │         拟凸函数              │
                │  (下水平集为凸)               │
                │                              │
                │  ┌──────────────────────┐    │
                │  │      凸函数          │    │
                │  │                      │    │
                │  │  ┌──────────────┐    │    │
                │  │  │ 对数凸函数   │    │    │
                │  │  │              │    │    │
                │  │  └──────────────┘    │    │
                │  │  ┌──────────────┐    │    │
                │  │  │ 对数凹函数   │    │    │
                │  │  │ (特例：凹)   │    │    │
                │  │  └──────────────┘    │    │
                │  └──────────────────────┘    │
                └──────────────────────────────┘
```

注：对数凸函数 ⊆ 凸函数，且对数凹函数未必是凸函数。
