# 弱对偶性与强对偶性

弱对偶与强对偶是[[对偶理论]]中最核心的两个对偶关系。弱对偶**恒成立**并提供下界；强对偶在凸优化中**条件成立**并给出精确等价。

## 一、弱对偶性 (Weak Duality)

### 1.1 定理

$$\boxed{d^* \leq p^*}$$

即对偶最优值永远不大于原问题最优值。

### 1.2 证明

对任意可行解 $\mathbf{x}$（$f_i(\mathbf{x}) \leq 0$, $h_i(\mathbf{x}) = 0$）和对偶可行 $(\boldsymbol{\lambda}, \boldsymbol{\nu})$（$\boldsymbol{\lambda} \succeq \mathbf{0}$）：

$$\begin{aligned} g(\boldsymbol{\lambda}, \boldsymbol{\nu}) &= \inf_{\mathbf{z}} \left(f_0(\mathbf{z}) + \sum \lambda_i f_i(\mathbf{z}) + \sum \nu_i h_i(\mathbf{z})\right) \\ &\leq f_0(\mathbf{x}) + \sum \lambda_i f_i(\mathbf{x}) + \sum \nu_i h_i(\mathbf{x}) \\ &\leq f_0(\mathbf{x}) \quad (\text{因为 } \lambda_i \geq 0, f_i(\mathbf{x}) \leq 0, h_i(\mathbf{x}) = 0) \end{aligned}$$

对 $\mathbf{x}$ 取下确界得 $g(\boldsymbol{\lambda}, \boldsymbol{\nu}) \leq p^*$，再对 $(\boldsymbol{\lambda}, \boldsymbol{\nu})$ 取上确界得 $d^* \leq p^*$。

### 1.3 关键性质

- **不需要凸性假设**（对任何优化问题成立）
- 对偶间隙 $\Delta = p^* - d^* \geq 0$

## 二、强对偶性 (Strong Duality)

### 2.1 定义

$$\boxed{d^* = p^*}$$

强对偶成立时，对偶问题与原问题**等价**：可以通过求解对偶问题来精确恢复原问题的最优值。

### 2.2 强对偶的重要性

- 对偶问题常比原问题**简单**（如对偶变量维度低、约束简单）
- 提供**停止准则**：$f_0(\mathbf{x}^{(k)}) - g(\boldsymbol{\lambda}^{(k)}) \leq \epsilon$
- 对偶变量给出**灵敏度信息**（影子价格）
- [[KKT条件]]成为最优性的充要条件

### 2.3 强对偶何时成立？

**非凸问题**：通常 $d^* < p^*$（有正的对偶间隙）。

**凸优化问题**：一般 $d^* = p^*$（在[[Slater条件与约束品性|约束品性]]下）。

---

## 三、对偶间隙

### 3.1 定义

$$\Delta_{\text{duality}} = p^* - d^* \geq 0$$

### 3.2 间隙为零的情况

- 凸问题 + Slater条件（或其他约束品性）
- 某些非凸问题也有零对偶间隙（如矩阵补全，在特定条件下）

### 3.3 间隙非零的情况

- 大多数非凸问题的SDP松弛（如Max-Cut的SDP松弛间隙约为12.2%）
- 整数规划的对偶间隙
- 凸问题但约束品性不满足

## 四、对偶在算法中的应用

### 4.1 停止准则

对任何迭代方法，若产生原始迭代 $\mathbf{x}^{(k)}$ 和对偶迭代 $(\boldsymbol{\lambda}^{(k)}, \boldsymbol{\nu}^{(k)})$：

$$f_0(\mathbf{x}^{(k)}) - g(\boldsymbol{\lambda}^{(k)}, \boldsymbol{\nu}^{(k)}) \leq f_0(\mathbf{x}^{(k)}) - p^*$$

上界给出了当前解与最优值的差距估计。

**典型停止条件**：

$$f_0(\mathbf{x}^{(k)}) - g(\boldsymbol{\lambda}^{(k)}, \boldsymbol{\nu}^{(k)}) \leq \epsilon$$

### 4.2 对偶提供了下界

任何对偶可行解提供了 $p^*$ 的下界：$g(\boldsymbol{\lambda}, \boldsymbol{\nu}) \leq p^*$。这对于分支定界等全局优化方法至关重要。

### 4.3 对偶上升法

直接通过最大化 $g(\boldsymbol{\lambda}, \boldsymbol{\nu})$ 求解对偶问题。特别当对偶问题具有可分解性时高效。

详见 [[近端梯度法与ADMM]]

## 五、强对偶与凸分析

### 5.1 几何观点

令 $\mathcal{G}$ 为可达值集（$(\mathbf{u}, \mathbf{v}, t)$ 其中 $\mathbf{u}$ 为不等式值，$\mathbf{v}$ 为等式值，$t$ 为目标值）。

$p^*$ 对应 $\mathcal{G}$ 在满足 $\mathbf{u} \preceq \mathbf{0}, \mathbf{v} = \mathbf{0}$ 时的最小 $t$。

Lagrange对偶对应用非垂超平面**支撑** $\mathcal{G}$ 的下界。

**强对偶** ⇔ 存在非垂支撑超平面恰好触及 $p^*$ 的点。

凸性 ⇒ $\mathcal{G}$ 在某种意义上"凸" ⇒ 存在支撑超平面 ⇒ 强对偶。

---

## 六、弱对偶与松弛

### 6.1 Lagrange松弛

Lagrange对偶本质上是一个"松弛"：

$$p^* = \min_{\mathbf{x}} \sup_{\boldsymbol{\lambda} \succeq \mathbf{0}, \boldsymbol{\nu}} L(\mathbf{x}, \boldsymbol{\lambda}, \boldsymbol{\nu}) \quad \text{(原问题)}$$

$$d^* = \sup_{\boldsymbol{\lambda} \succeq \mathbf{0}, \boldsymbol{\nu}} \inf_{\mathbf{x}} L(\mathbf{x}, \boldsymbol{\lambda}, \boldsymbol{\nu}) \quad \text{(对偶问题)}$$

弱对偶 = suprinf ≤ infsup（max-min不等式，恒成立）。

强对偶 = suprinf = infsup（鞍点性质，凸+约束品性下成立）。
