# 保凸运算与复合规则

保凸运算告诉我们：用凸函数**构造**新凸函数的规则。这是[[凸优化问题标准形式|建模]]中判断函数凸性的最实用工具。

## 一、基本保凸运算

### 1.1 非负加权和

若 $f_1, \ldots, f_m$ 为凸函数，$w_i \geq 0$，则：

$$f(\mathbf{x}) = \sum_{i=1}^{m} w_i f_i(\mathbf{x}) \text{ 是凸函数}$$

**无限版本**：若 $f(\mathbf{x}, \mathbf{y})$ 对每固定的 $\mathbf{y}$ 为 $\mathbf{x}$ 的凸函数，$w(\mathbf{y}) \geq 0$，则：

$$\int_{\mathcal{Y}} w(\mathbf{y}) f(\mathbf{x}, \mathbf{y}) \, d\mathbf{y}$$

在积分有限的条件下是凸函数。

### 1.2 仿射变量复合

若 $g$ 凸，则 $f(\mathbf{x}) = g(\mathbf{A}\mathbf{x} + \mathbf{b})$ 也是凸函数。

> **直觉**：$\mathbf{x} \mapsto \mathbf{A}\mathbf{x} + \mathbf{b}$ 是线性变换，不会破坏凸性。

### 1.3 逐点最大与逐点上确界

若 $f_1, \ldots, f_m$ 为凸函数，则：

$$f(\mathbf{x}) = \max\{f_1(\mathbf{x}), \ldots, f_m(\mathbf{x})\} \text{ 是凸函数}$$

**无限族版本**：若对每个 $\mathbf{y} \in \mathcal{A}$，$f(\mathbf{x}, \mathbf{y})$ 为 $\mathbf{x}$ 的凸函数，则：

$$g(\mathbf{x}) = \sup_{\mathbf{y} \in \mathcal{A}} f(\mathbf{x}, \mathbf{y}) \text{ 是凸函数}$$

**关键应用**：
- 分段线性凸函数：$f(x) = \max_i (a_i x + b_i)$
- [[共轭函数]]：$f^*(\mathbf{y}) = \sup_{\mathbf{x}} (\mathbf{y}^T \mathbf{x} - f(\mathbf{x}))$ 恒为凸
- 最大特征值：$\lambda_{\max}(\mathbf{X}) = \sup_{\|\mathbf{v}\|_2 = 1} \mathbf{v}^T \mathbf{X}\mathbf{v}$（凸）

### 1.4 部分最小化

若 $f(\mathbf{x}, \mathbf{y})$ 是 $(\mathbf{x}, \mathbf{y})$ 的联合凸函数，$C$ 为非空凸集，则：

$$g(\mathbf{x}) = \inf_{\mathbf{y} \in C} f(\mathbf{x}, \mathbf{y}) \text{ 是凸函数}$$

（要求下确界有限）

**关键应用**：Schur补引理 —— $\mathbf{X} = \begin{bmatrix} \mathbf{A} & \mathbf{B} \\ \mathbf{B}^T & \mathbf{C} \end{bmatrix} \succeq 0$（$\mathbf{C} \succ 0$）当且仅当 $\mathbf{A} - \mathbf{B}\mathbf{C}^{-1}\mathbf{B}^T \succeq 0$。右端实为二次型的部分最小化的结果。

### 1.5 透视函数复合

若 $f$ 为凸函数，则：

$$g(\mathbf{x}, t) = t f\left(\frac{\mathbf{x}}{t}\right), \quad \text{dom }g = \{(\mathbf{x}, t) \mid \mathbf{x}/t \in \text{dom }f, t > 0\}$$

是凸函数。

**应用**：
- $f(x) = x^T x / t$（$t > 0$）是凸的 ⇒ [[二次规划 QP|QCQP]]可转化为SOCP
- KL散度 $u\log(u/v)$：取 $f(x) = x\log x$ 和 $g(u, v) = v f(u/v) = u\log(u/v)$

## 二、标量复合规则

$h: \mathbb{R} \to \mathbb{R}$，$g: \mathbb{R}^n \to \mathbb{R}$，$f(\mathbf{x}) = h(g(\mathbf{x}))$。

### 2.1 规则

$n = 1$（一维输入）时，$f''(x) = h''(g(x)) \cdot [g'(x)]^2 + h'(g(x)) \cdot g''(x)$。

| $h$ 的凸性/单调性 | $g$ 的凸性 | $f = h \circ g$ |
|------------------|-----------|----------------|
| $h$ 凸且**非减** | $g$ 凸 | 凸 |
| $h$ 凸且**非增** | $g$ 凹 | 凸 |
| $h$ 凹且**非减** | $g$ 凹 | 凹 |
| $h$ 凹且**非增** | $g$ 凸 | 凹 |

### 2.2 关键例

- $e^{g(x)}$：$h(x)=e^x$（凸、非减）+ $g$ 凸 ⇒ 凸
- $-\log g(x)$：$h(x)=-\log x$（凸、非减）+ $g$ 凹 ⇒ 凸（如 $-\log(-x)$ 在 $x < 0$）
- $\frac{1}{g(x)}$（$g > 0$）：$h(x)=1/x$（凸、非增）+ $g$ 凹 ⇒ 凸

## 三、向量复合规则

$f(\mathbf{x}) = h(g_1(\mathbf{x}), \ldots, g_k(\mathbf{x}))$。

设 $h: \mathbb{R}^k \to \mathbb{R}$，$g_i: \mathbb{R}^n \to \mathbb{R}$。

### 3.1 规则

若 $h$ 凸且对**每个**自变量**非减**，每个 $g_i$ 凸 ⇒ $f$ 凸。

### 3.2 关键例

- $\sum_i e^{g_i(\mathbf{x})}$：$h(\mathbf{z}) = \sum e^{z_i}$（凸、各分量非减）+ $g_i$ 凸
- $\log\left(\sum_i e^{g_i(\mathbf{x})}\right)$：$h(\mathbf{z}) = \log\sum e^{z_i}$（凸、各分量非减）+ $g_i$ 凸

1. 是否可分解为已知凸函数的非负加权和？
   ├─ 是 → 凸
   └─ 否 → 2

2. 是否为 g(Ax+b) 形式（g 凸）？
   ├─ 是 → 凸
   └─ 否 → 3

3. 是否为 max / sup 一族凸函数？
   ├─ 是 → 凸
   └─ 否 → 4
