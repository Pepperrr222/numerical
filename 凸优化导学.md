## 起点：我们面对的根本问题

给定一个函数 $f(\mathbf{x})$，我们想在约束条件下找到它的最小值。

$$\min f(\mathbf{x}) \quad \text{s.t.} \quad \mathbf{x} \text{ 满足某些条件}$$

两个核心焦虑：
1. 我找到的是**全局最小**吗？还是被卡在某个局部凹槽里？
2. 我能**有效地算出**这个最小值吗？

一般的 $f$ 和一般的约束下，这两个问题都很难。非凸函数的局部极小值数量可能随维度指数增长，且没有多项式时间算法能保证全局最优。

**凸优化的答案**：如果 $f$ 是[[凸函数]]，约束集是[[凸集]]，那么任意局部最小点就是全局最小点，且存在多项式时间算法（内点法）求解。这就是我们要研究凸优化的根本动机。

---

## 第一步：约束集——"我们在哪里找"

先放下目标函数，只看**可行域** $\mathcal{F}$。我们的搜索范围必须是一个"好"的集合。

**核心直觉**：连接集合中任意两点的线段，不应跑出集合之外。

这就引出了[[凸集定义与性质|凸集的定义]]：

$$C \text{ 是凸集} \iff \forall \mathbf{x}, \mathbf{y} \in C, \forall \theta \in [0,1]: \theta\mathbf{x} + (1-\theta)\mathbf{y} \in C$$

### 为什么线段必须在集合内？

这个要求看似简单，但它是整个理论的地基。考虑下面的推理链：

1. 我们想用**迭代算法**找最优解
2. 每一步迭代产生一个点 $\mathbf{x}^{(k)}$
3. 如果我们取两个可行点的加权平均（比如做平均/插值），这个新点必须仍然可行
4. 凸性保证了：从任意可行点出发，向任意其他可行点方向走一小步，永远不越界

这就是凸性在**算法层面**的根本意义——你的搜索空间没有"空洞"。

### 从凸集能做什么？

一旦有了凸集，我们立即可以问：两个不相交的凸集能被一个超平面分开吗？

答案是[[分离超平面定理]]：$\mathbf{a}^T \mathbf{x} \leq b$（一侧）和 $\mathbf{a}^T \mathbf{x} \geq b$（另一侧）。

**这条定理为什么重要？** 因为它为后续的所有对偶理论提供了几何基础——Lagrange乘子本质上就是一个分离超平面的法向量。我们会在后面的"对偶理论"一节看到这个联系。

在边界点处，分离超平面退化为[[支撑超平面]]——这直接对应凸函数的**切线全局下界**性质（一阶条件）。

### 保凸运算：构造复杂凸集的工具

实际问题中，我们的可行域是由多个约束**同时**定义的：
$$f_1(\mathbf{x}) \leq 0 \text{ 且 } f_2(\mathbf{x}) \leq 0 \text{ 且 } \ldots$$

每个 $f_i(\mathbf{x}) \leq 0$ 定义了一个子水平集。我们需要的可行域是这些子水平集的**交集**。

[[保凸运算]]告诉我们：凸集的任意交仍是凸集。这保证了"同时满足多个约束"的可行域仍然是凸的。

类似地，仿射变换（线性映射、平移）也保凸——这在建模中极其常用（变量替换、去量纲化等）。

## 第二步：目标函数——"我们在优化什么"

现在有了可行的搜索范围（凸集），接下来要研究目标函数 $f$。

**核心直觉**：$f$ 的图形必须"向上弯"，不能有凹陷。

[[凸函数定义与基本性质|凸函数的定义]]：

$$f(\theta\mathbf{x} + (1-\theta)\mathbf{y}) \leq \theta f(\mathbf{x}) + (1-\theta)f(\mathbf{y})$$

### 这个定义到底在说什么？

函数图像上的弦，始终在函数图像的**上方**。换句话：函数值永远不超过线性插值。

**这就是"没有凹陷"的精确表述。** 如果函数有凹陷，弦会穿到函数图像之下。

### 从几何到分析：一阶条件和二阶条件

凸性的定义是纯粹几何的，不方便计算。我们需要分析条件。

**[[凸函数一阶条件]]**：若 $f$ 可微：

$$f(\mathbf{y}) \geq f(\mathbf{x}) + \nabla f(\mathbf{x})^T (\mathbf{y} - \mathbf{x})$$

函数图像在任意切线的**上方**。

这个条件的推导来自于[[支撑超平面]]定理：对 [[上镜图与下水平集|上镜图]] $\text{epi }f$（它是凸集 ⇔ $f$ 凸），在边界点 $(\mathbf{x}, f(\mathbf{x}))$ 处存在支撑超平面，而这个支撑超平面正是 $f$ 的一阶Taylor展开所定义的平面。

**这就是凸集与凸函数的桥梁**：上镜图。

**[[凸函数二阶条件]]**：若 $f$ 二阶可微：

$$\nabla^2 f(\mathbf{x}) \succeq 0$$

Hessian处处半正定 ⇔ 曲率非负 ⇔ "向上弯"。在 $\mathbb{R}$ 上就是 $f''(x) \geq 0$，非常直观。

### 为什么一阶条件导出全局最优性？

对于无约束优化 $\min f(\mathbf{x})$，由一阶条件：

$$f(\mathbf{y}) \geq f(\mathbf{x}^*) + \nabla f(\mathbf{x}^*)^T (\mathbf{y} - \mathbf{x}^*)$$

如果 $\nabla f(\mathbf{x}^*) = \mathbf{0}$，那么对所有 $\mathbf{y}$：$f(\mathbf{y}) \geq f(\mathbf{x}^*)$。

**这就是"局部最小 ⇒ 全局最小"的严格证明。** 一个一阶条件就搞定了。

### Jensen不等式：凸函数的概率含义

[[Jensen不等式]]：$f(\mathbb{E}[X]) \leq \mathbb{E}[f(X)]$

这是凸函数定义的**连续/概率版本**，我们后面会在对偶理论中再次遇到它（对偶函数是凹函数的本质原因就是"期望的下界 ≤ 下界的期望"类型的不等式）。

### 共轭函数：对偶理论的准备

[[共轭函数]]：$f^*(\mathbf{y}) = \sup_{\mathbf{x}} (\mathbf{y}^T\mathbf{x} - f(\mathbf{x}))$

**为什么现在引入共轭？** 因为它在对偶理论中发挥关键作用——Lagrange对偶函数可以用Fenchel共轭表达：

$$g(\boldsymbol{\nu}) = -\mathbf{b}^T\boldsymbol{\nu} - f_0^*(-\mathbf{A}^T\boldsymbol{\nu})$$

共轭函数**恒为凸**（因为是仿射函数的逐点上确界）——这是[[保凸运算与复合规则|保凸运算]]中最重要的结论之一。

共轭还提供了分析上的方便：$\nabla f$ 和 $\nabla f^*$ 互为反函数（当 $f$ 严格凸时）。更多细节见 [[共轭函数]]。

### 放宽凸性：拟凸与对数凸

并非所有有用的函数都满足严格的凸性。[[拟凸函数与对数凸函数]]提供了两种有意义的推广：
- **拟凸**：下水平集为凸 ⇒ 可通过二分法求解
- **对数凸**：$\log f$ 为凸 ⇒ 出现在[[几何规划]]中

## 第三步：把集合和函数拼起来——凸优化问题

现在我们有：
- 凸可行域 $\mathcal{F}$（凸约束定义）
- 凸目标函数 $f_0$

把它们放在一起就构成了[[凸优化问题标准形式]]：

$$\min f_0(\mathbf{x}) \quad \text{s.t.} \quad f_i(\mathbf{x}) \leq 0, \quad \mathbf{A}\mathbf{x} = \mathbf{b}$$

其中 $f_i$ 凸，等式约束仿射。

### 为什么等式约束必须是仿射？

如果 $h(\mathbf{x}) = 0$ 且 $h$ 非线性，可行集 $\{\mathbf{x} \mid h(\mathbf{x}) = 0\}$ 一般**非凸**（例如 $x^2 + y^2 = 1$ 定义的圆环，内部是空洞）。

所以凸优化的标准形式中，等式约束只能是 $\mathbf{A}\mathbf{x} = \mathbf{b}$。

### 从简单到复杂：问题家族的包含链

这个链是一个不断放松约束形式的过程，每一步都扩展了建模能力：

$$\text{LP} \subset \text{QP} \subset \text{QCQP} \subset \text{SOCP} \subset \text{SDP}$$

**[[线性规划 LP]]**：目标与约束全为仿射。可行域为多面体，最优解在顶点取得。单纯形法利用了这个几何结构：最优解必在某个顶点，只需要沿着棱从一个顶点移向更好的相邻顶点。

**[[二次规划 QP]]**：目标换成二次型 $\frac{1}{2}\mathbf{x}^T\mathbf{P}\mathbf{x} + \mathbf{q}^T\mathbf{x}$（$\mathbf{P} \succeq 0$），约束仍为仿射。Markowitz投资组合优化就是QP。当 $\mathbf{P} \succ 0$ 时无约束QP有解析解：$\mathbf{x}^* = -\mathbf{P}^{-1}\mathbf{q}$。

**[[二次约束二次规划 QCQP]]**：约束也变成二次型。可行域从多面体变成了**椭球的交**。每个约束 $\frac{1}{2}\mathbf{x}^T\mathbf{P}_i\mathbf{x} + \mathbf{q}_i^T\mathbf{x} + r_i \leq 0$（$\mathbf{P}_i \succeq 0$）定义了一个实心椭球。

**[[二阶锥规划 SOCP]]**：将 QCQP 的二次约束表达成锥约束。$\|\mathbf{A}\mathbf{x} + \mathbf{b}\|_2 \leq \mathbf{c}^T\mathbf{x} + d$。二阶锥 $\{(\mathbf{y}, t) \mid \|\mathbf{y}\|_2 \leq t\}$ 是自对偶真锥，SOCP在此框架下有极好的理论性质。

**[[半定规划 SDP]]**：最一般的形式。约束为线性矩阵不等式（LMI）：$\mathbf{F}_0 + \sum x_i\mathbf{F}_i \preceq \mathbf{0}$。半正定锥 $\mathbf{S}_+^k$ 是自对偶真锥。SDP包含了SOCP（通过Schur补引理转化为LMI），并能表达特征值优化、矩阵范数、核范数松弛等问题。

[[几何规划]]是一个特例：在原变量下**非凸**，但通过对数变换 $y_i = \log x_i$ 后可凸化。这告诉我们：有些看似非凸的问题，通过巧妙的变量变换可以"凸化"。

## 第四步：怎么判断找到的是最优解？——对偶与KKT

现在我们知道怎么"写"一个凸优化问题。但**给定候选解 $\tilde{\mathbf{x}}$，怎么判断它是最优的？**

这就是[[对偶理论]]要回答的问题。

### Lagrange对偶：将"硬约束"变成"软罚"

考虑 $\min f_0(\mathbf{x})$ s.t. $f_i(\mathbf{x}) \leq 0$。

我们可以问：如果把约束 $f_i \leq 0$ 变得不那么严格（允许违反，但付出代价），最优值会怎样变化？

定义[[Lagrange对偶函数]]：

$$L(\mathbf{x}, \boldsymbol{\lambda}) = f_0(\mathbf{x}) + \sum_{i=1}^{m} \lambda_i f_i(\mathbf{x})$$

$$g(\boldsymbol{\lambda}) = \inf_{\mathbf{x}} L(\mathbf{x}, \boldsymbol{\lambda})$$

**关键洞察**：对每个 $\boldsymbol{\lambda} \succeq \mathbf{0}$：

$$g(\boldsymbol{\lambda}) \leq p^*$$

因为对任何可行解 $\mathbf{x}$（$f_i(\mathbf{x}) \leq 0$）：

$$L(\mathbf{x}, \boldsymbol{\lambda}) = f_0(\mathbf{x}) + \underbrace{\sum \lambda_i f_i(\mathbf{x})}_{\leq 0} \leq f_0(\mathbf{x})$$

然后取 $\inf_{\mathbf{x}}$ 即得 $g(\boldsymbol{\lambda}) \leq p^*$。

**这就是[[弱对偶性与强对偶性|弱对偶性]]：对偶函数给出原问题最优值的全局下界。** 这条恒成立，无需任何假设。

### 强对偶：下界恰好等于最优值

有了弱对偶，我们想知道：这个下界能不能**达到**最优值？

$\sup_{\boldsymbol{\lambda} \succeq \mathbf{0}} g(\boldsymbol{\lambda}) = p^*$ 称为**强对偶**。

什么时候成立？[[Slater条件与约束品性|Slater条件]]给出了凸优化的充分条件：

> 存在一个点 $\tilde{\mathbf{x}}$ 使得所有非线性不等式约束**严格**成立：$f_i(\tilde{\mathbf{x}}) < 0$。

**几何意义**：可行域有"内部"（相对内部非空），使得在最优值 $p^*$ 处，[[分离超平面定理|分离超平面]]可以是非垂直的——而非垂直的超平面正好对应Lagrange乘子的有限值。

这里可以看出整个理论的线索：
1. [[凸集]] → [[分离超平面定理]] → 存在分离超平面
2. [[分离超平面定理]] → [[Lagrange对偶函数]] → 乘子作为超平面法向量
3. Slater条件 → 保证超平面非垂直（乘子有限）→ 强对偶

### KKT条件：最优性的充要条件

强对偶成立后，[[KKT条件]]给出可微凸优化问题最优解的**充要条件**：

1. **原始可行**：$f_i(\tilde{\mathbf{x}}) \leq 0$, $h_i(\tilde{\mathbf{x}}) = 0$
2. **对偶可行**：$\tilde{\lambda}_i \geq 0$
3. **互补松弛**：$\tilde{\lambda}_i f_i(\tilde{\mathbf{x}}) = 0$
4. **驻点**：$\nabla f_0 + \sum \tilde{\lambda}_i \nabla f_i + \sum \tilde{\nu}_i \nabla h_i = \mathbf{0}$

**互补松弛性的意义**：$\tilde{\lambda}_i > 0 \iff f_i(\tilde{\mathbf{x}}) = 0$（约束是**紧的**）。只有紧约束影响最优解——这个直觉非常重要。

**驻点条件的意义**：目标梯度可表示为约束梯度的**非负线性组合**。几何上：$-\nabla f_0(\tilde{\mathbf{x}})$ 位于紧约束梯度生成的**锥**中——这是[[支撑超平面|法锥]]和[[分离超平面定理]]的直接应用。

### 灵敏度：对偶变量的经济学含义

[[扰动与灵敏度分析]]告诉我们：

$$\frac{\partial p^*}{\partial u_i} \approx -\lambda_i^*$$

- $\lambda_i^*$ 就是约束 $f_i$ 的**影子价格**
- $\lambda_i^*$ 大 ⇒ 该约束是瓶颈，放松它收益大
- $\lambda_i^* = 0$ ⇒ 该约束不影响最优解

这个解释在 [[线性规划 LP|LP]] 中最早被广泛使用（资源的边际价值），但通过[[对偶理论]]推广到了所有凸优化问题。

## 第五步：怎么真正算出最优解？——算法

理论讲完了。现在我们**真的需要算出 $\mathbf{x}^*$**。

[[凸优化算法]]的所有方法都是基于前面的理论基础。

### 无约束光滑优化：梯度告诉你方向

[[无约束凸优化算法]]从一阶条件出发：$\nabla f(\mathbf{x}^*) = \mathbf{0}$。所以我们需要驱动梯度趋零。

**梯度下降**：$\mathbf{x}^{(k+1)} = \mathbf{x}^{(k)} - t \nabla f(\mathbf{x}^{(k)})$

为什么有效？由[[凸函数一阶条件]]的"二次上界"（Lipschitz梯度）：
$$f(\mathbf{x}^{k+1}) \leq f(\mathbf{x}^{k}) - \frac{1}{2L}\|\nabla f\|^2$$

每一步保证**充分下降**，收敛率 $O(1/k)$。

**Nesterov加速**：加入"动量"（惯性），收敛率提升至 $O(1/k^2)$——这是光滑凸优化的**理论最优**速率。

**Newton法**：利用[[凸函数二阶条件|Hessian信息]]做二阶近似：
$$\mathbf{x}^{(k+1)} = \mathbf{x}^{(k)} - [\nabla^2 f]^{-1} \nabla f$$

在局部进入**二次收敛**——每步有效数字翻倍。但每步 $O(n^3)$ 计算Hessian逆，仅适合中小规模。

**L-BFGS**：用有限内存近似Hessian逆，仅需 $O(mn)$ 存储（$m \approx 5-20$），在实践中是中等规模问题的最佳折中。

### 有约束：把约束"吸收"掉

[[有约束凸优化算法]]的核心思想是：将约束优化转化为一系列无约束优化。

**障碍法**：在目标函数后加上**对数障碍**：
$$\min t f_0(\mathbf{x}) - \sum \log(-f_i(\mathbf{x}))$$

$\log(-f_i)$ 在约束边界处 $\to +\infty$，形成"壁垒"阻止越界。

**中心路径** $\mathbf{x}^*(t)$ 随 $t \to \infty$ 趋于真最优解 $\mathbf{x}^*$。每步用Newton法求解障碍子问题。

**原始-对偶内点法**：不精确求解子问题，而是对**扰动KKT条件**（$\lambda_i f_i = -1/t$）直接应用Newton步，同时更新原始和对偶变量。这是目前实际应用中最快的方法。

复杂度：$O(\sqrt{m} \log(1/\epsilon))$ 次Newton迭代——多项式时间。

### 非光滑凸优化：次梯度与近端方法

如果函数不可微（如 $\ell_1$ 范数、hinge loss），梯度不存在。

**[[次梯度方法]]**：用次梯度 $\mathbf{g} \in \partial f(\mathbf{x})$ 代替梯度：

$$f(\mathbf{y}) \geq f(\mathbf{x}) + \mathbf{g}^T (\mathbf{y} - \mathbf{x})$$

次梯度**不保证下降方向**！所以步长必须预先确定（不可线搜索），收敛率只有 $O(1/\sqrt{k})$。

**[[近端梯度法与ADMM|近端梯度法]]**：利用复合结构 $f = g + h$（$g$ 光滑，$h$ 非光滑但近端易算）：

$$\mathbf{x}^{(k+1)} = \text{prox}_{t h}(\mathbf{x}^{(k)} - t\nabla g)$$

近端算子 $\text{prox}$ 对 $\ell_1$ 范数给出**软阈值**闭式解。收敛率 $O(1/k)$（FISTA加速到 $O(1/k^2)$）——比次梯度法快得多。

**[[近端梯度法与ADMM|ADMM]]**：处理带线性耦合约束的可分问题：
$$\min f(\mathbf{x}) + g(\mathbf{z}) \quad \text{s.t.} \quad \mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{z} = \mathbf{c}$$

三步交替：$\mathbf{x}$-子问题 → $\mathbf{z}$-子问题 → 对偶乘子更新 $\mathbf{y}^{(k+1)} = \mathbf{y}^{(k)} + \rho(\mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{z} - \mathbf{c})$。

对偶乘子更新的本质是**对偶上升法**——增量正好是约束违反量乘 $\rho$。

ADMM将[[对偶理论|对偶分解]]与[[有约束凸优化算法|增广Lagrange法]]的优点结合：可分解并行求解（对偶分解的优点），且对非光滑问题也收敛（增广项的优点）。

### 大规模优化：随机梯度

[[随机梯度下降 SGD]]：当 $f(\mathbf{x}) = \frac{1}{N}\sum f_i(\mathbf{x})$ 且 $N$ 巨大时：

$$\mathbf{x}^{(k+1)} = \mathbf{x}^{(k)} - t_k \nabla f_{i_k}(\mathbf{x}^{(k)})$$

每次只用**一个随机样本**的梯度（$O(1)$ 计算而非 $O(N)$）。

SGD的收敛率仅 $O(1/\sqrt{k})$（由于随机梯度的方差），但计算节省的常数因子在大规模场景下碾压全梯度方法。

**Adam**等自适应方法通过**动量**（惯性）+ **自适应学习率**（每维缩放）使得实践中收敛极快，是深度学习的事实标准。

## 全线索回顾

整条线索可以用一个紧凑的逻辑链表达：

```
[动机] 我们要全局最优解 + 高效算法
   ↓
[凸集] 可行域不能有凹陷 → 线段在集合内
   ↓
[保凸运算] 多个约束的交仍是凸集 → 可以建模复杂可行域
   ↓
[上镜图] 凸函数 ⇔ 其上镜图是凸集 → 凸集理论可用于函数
   ↓
[一阶/二阶条件] Hessian ≥ 0, 切线全局下界 → 验证凸性的分析工具
   ↓
[凸优化问题] 凸函数 + 凸约束 → 局部最优 = 全局最优
   ↓
[Lagrange对偶] 约束"价格化" → 对偶函数提供 p* 的下界
   ↓
[Slater + 强对偶] 有严格内部点 → 对偶间隙为零
   ↓
[KKT条件] 强对偶 + 可微 → 最优性充要条件（四个方程）
   ↓
[算法] 求解KKT(内点法) 或 直接最小化目标(梯度法/Newton/ADMM)
```

**每一条依赖链**：
- `凸集 → 保凸运算 → 建模复杂可行域`
- `凸集 → 上镜图 → 凸函数等价于上镜图为凸集`
- `凸集 → 分离超平面 → Lagrange乘子 → 对偶理论 → KKT`
- `凸函数 → 一阶条件 → ∇f(x*) = 0 ⇒ 全局最小`
- `凸函数 → 一阶/二阶条件 → 梯度单调性 → 梯度下降收敛`
- `凸函数 → 共轭函数 → Fenchel对偶 → Lagrange对偶的等价视角`
- `对偶 → 弱对偶 → 强对偶(Slater) → KKT → 内点法求解KKT`
- `对偶 → 灵敏度 → 影子价格 → 经济学解释`
- `算法 → 障碍法 → 内点法 → 多项式复杂度保证`
