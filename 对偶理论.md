# 对偶理论 (Duality Theory)

[[Lagrange对偶]]是凸优化的理论核心。它不仅提供最优值的下界，还在凸条件下给出**强对偶**（零对偶间隙），由此导出 [[KKT条件]]。

## 一、[[Lagrange对偶函数]]

### 1.1 Lagrange函数

考虑标准形式的[[凸优化问题标准形式|优化问题]]：

$$\begin{aligned} \min \quad & f_0(\mathbf{x}) \\ \text{s.t.} \quad & f_i(\mathbf{x}) \leq 0, \quad i=1,\ldots,m \\ & h_i(\mathbf{x}) = 0, \quad i=1,\ldots,p \end{aligned}$$

**Lagrange函数**：

$$L(\mathbf{x}, \boldsymbol{\lambda}, \boldsymbol{\nu}) = f_0(\mathbf{x}) + \sum_{i=1}^{m} \lambda_i f_i(\mathbf{x}) + \sum_{i=1}^{p} \nu_i h_i(\mathbf{x})$$

其中 $\boldsymbol{\lambda} \succeq 0$（$\lambda_i \geq 0$），$\boldsymbol{\nu}$ 无限制。

> $L(\mathbf{x}, \boldsymbol{\lambda}, \boldsymbol{\nu})$ 是 $\mathbf{x}$ 的**仿射族**，由 $f_i$ 和 $h_i$ 的线性组合构成。

### 1.2 Lagrange对偶函数

$$g(\boldsymbol{\lambda}, \boldsymbol{\nu}) = \inf_{\mathbf{x} \in \mathcal{D}} L(\mathbf{x}, \boldsymbol{\lambda}, \boldsymbol{\nu})$$

其中 $\mathcal{D} = \bigcap \text{dom }f_i \cap \bigcap \text{dom }h_i$。

### 1.3 对偶函数的基本性质

1. **$g$ 恒为凹函数**：$g$ 是关于 $(\boldsymbol{\lambda}, \boldsymbol{\nu})$ 的**仿射函数族**的逐点下确界，从而恒为凹函数
2. **下界性质**：对任意 $\boldsymbol{\lambda} \succeq \mathbf{0}$ 和任意 $\boldsymbol{\nu}$：
   $$g(\boldsymbol{\lambda}, \boldsymbol{\nu}) \leq p^*$$
   即对偶函数给出原问题最优值的**全局下界**

## 二、Lagrange对偶问题

### 2.1 对偶问题的形式

$$\begin{aligned} \max_{\boldsymbol{\lambda}, \boldsymbol{\nu}} \quad & g(\boldsymbol{\lambda}, \boldsymbol{\nu}) \\ \text{s.t.} \quad & \boldsymbol{\lambda} \succeq 0 \end{aligned}$$

- 对偶问题是**凸优化问题**（最大化凹函数 $g$）
- 对偶最优值记为 $d^*$

### 2.2 对偶问题的意义

| | 原问题 (Primal) | 对偶问题 (Dual) |
|---|---|---|
| 变量 | $\mathbf{x} \in \mathbb{R}^n$ | $(\boldsymbol{\lambda}, \boldsymbol{\nu}) \in \mathbb{R}^m \times \mathbb{R}^p$ |
| 目标 | $\min f_0$（凸） | $\max g$（凹 → 凸） |
| 约束 | $f_i \leq 0$, $h_i = 0$ | $\boldsymbol{\lambda} \succeq 0$（简单） |
| 最优值 | $p^*$ | $d^*$ |

---

## 三、[[弱对偶性与强对偶性]]

### 3.1 弱对偶性

$$\boxed{d^* \leq p^*}$$

**弱对偶恒成立**（不需要凸性假设）。对偶间隙 $\Delta = p^* - d^* \geq 0$。

> 直观：对偶问题通过"松弛"约束给出了原问题的下界估计。

### 3.2 强对偶性

若 $d^* = p^*$（**零对偶间隙**），则称**强对偶**成立。

强对偶 ⇒ 可以从对偶问题**精确恢复**原最优值。

### 3.3 强对偶不成立的例子

- 非凸问题通常 $d^* < p^*$
- 凸问题中也可能强对偶不成立（罕见，需约束品性）

---

## 四、[[Slater条件与约束品性]]

### 4.1 Slater条件（凸优化的强对偶充分条件）

对于凸优化问题（$f_i$ 凸，$h_i$ 仿射），若存在 $\tilde{\mathbf{x}} \in \text{relint }\mathcal{D}$ 满足：

$$f_i(\tilde{\mathbf{x}}) < 0, \quad i=1,\ldots,m, \qquad \mathbf{A}\tilde{\mathbf{x}} = \mathbf{b}$$

则**强对偶成立**，且若 $d^* > -\infty$，则对偶最优值可达到。

**放松Slater条件**：若前 $k$ 个约束为仿射不等式，只需 $f_i(\tilde{\mathbf{x}}) \leq 0$（$i=1,\ldots,k$），其余约束严格成立。

### 4.2 线性规划的强对偶

LP 中 Slater 条件简化为**可行性**：若原问题（或对偶问题）可行，则强对偶成立。

### 4.3 其他约束品性

- **Mangasarian-Fromovitz约束品性**（MFCQ）— NLP通用
- **线性独立约束品性**（LICQ）— NLP局部条件
- **Abadie约束品性**— 基于切线锥
- 凸优化中，**Slater条件是应用最广的约束品性**

---

## 五、[[KKT条件]]

KKT 条件（Karush-Kuhn-Tucker条件）是凸优化中**最优性的充要条件**（在强对偶成立时）。

### 5.1 四个条件

假设 $f_i, h_i$ 可微且强对偶成立。$\tilde{\mathbf{x}}$ 和 $(\tilde{\boldsymbol{\lambda}}, \tilde{\boldsymbol{\nu}})$ 为原始-对偶最优解 **当且仅当**：

**(1) 原始可行性**：
$$f_i(\tilde{\mathbf{x}}) \leq 0, \quad h_i(\tilde{\mathbf{x}}) = 0$$

**(2) 对偶可行性**：
$$\tilde{\lambda}_i \geq 0$$

**(3) 互补松弛性**：
$$\boxed{\tilde{\lambda}_i f_i(\tilde{\mathbf{x}}) = 0, \quad i = 1,\ldots,m}$$

**(4) 驻点条件**：
$$\boxed{\nabla f_0(\tilde{\mathbf{x}}) + \sum_{i=1}^{m} \tilde{\lambda}_i \nabla f_i(\tilde{\mathbf{x}}) + \sum_{i=1}^{p} \tilde{\nu}_i \nabla h_i(\tilde{\mathbf{x}}) = \mathbf{0}}$$

### 5.2 KKT条件的解释

| 条件 | 含义 |
|------|------|
| 原始可行性 | 解必须在可行域内 |
| 对偶可行性 | Lagrange乘子必须非负 |
| 互补松弛性 | 只有**紧约束**（$f_i=0$）才能有正的乘子；松弛约束 $\lambda_i=0$ |
| 驻点条件 | 目标函数梯度是约束梯度的**非负线性组合** |

### 5.3 凸优化中的KKT定理

若 $f_i$ 可微凸函数且**强对偶成立**（如满足Slater条件），则：
- $\tilde{\mathbf{x}}$ 为原问题最优解 **当且仅当** 存在 $(\tilde{\boldsymbol{\lambda}}, \tilde{\boldsymbol{\nu}})$ 使得KKT条件成立
- KKT条件是充要条件，而非仅仅是必要条件

## 六、[[扰动与灵敏度分析]]

### 6.1 扰动问题

考虑逐步松弛/收紧约束后的最优值：

$$p^*(\mathbf{u}, \mathbf{v}) = \inf_{\mathbf{x}} \left\{f_0(\mathbf{x}) \mid f_i(\mathbf{x}) \leq u_i, h_i(\mathbf{x}) = v_i\right\}$$

注意 $p^*(\mathbf{0}, \mathbf{0}) = p^*$。

### 6.2 灵敏度解释

当强对偶成立且对偶最优解 $(\boldsymbol{\lambda}^*, \boldsymbol{\nu}^*)$ 唯一时：

$$\frac{\partial p^*(\mathbf{u}, \mathbf{v})}{\partial u_i}\bigg|_{\mathbf{u}=\mathbf{0}} \approx -\lambda_i^*$$
$$\frac{\partial p^*(\mathbf{u}, \mathbf{v})}{\partial v_i}\bigg|_{\mathbf{v}=\mathbf{0}} \approx -\nu_i^*$$

**经济学意义**：
- $\lambda_i^*$：约束 $f_i$ 每收紧一个单位，最优值的**改进量**（影子价格）
- 大 $\lambda_i^*$ ⇒ 该约束是瓶颈
- $\lambda_i^* = 0$ ⇒ 该约束松弛，不影响最优值

## 七、对偶在算法中的应用

对偶理论直接指导凸优化算法设计：

1. **内点法**：求解KKT条件的扰动版本（见 [[有约束凸优化算法]]）
2. **对偶上升法**：更新 $\boldsymbol{\lambda}$ 以最大化 $g(\boldsymbol{\lambda})$（见 [[近端梯度法与ADMM]]）
3. **停止准则**：$f_0(\mathbf{x}^{(k)}) - g(\boldsymbol{\lambda}^{(k)}) \leq \epsilon$ 作为精度度量
4. **对偶分解**：可分结构的问题，对偶分解为子问题并行求解

## 八、对偶理论的核心框架图

```
原问题 (Primal)                    对偶问题 (Dual)
min f₀(x)                           max g(λ,ν)
s.t. fᵢ(x) ≤ 0          ←──→       s.t. λ ≥ 0
     Ax = b                              (无约束)
     
       弱对偶: d* ≤ p* (恒成立)
       强对偶: d* = p* (凸 + Slater)
       
       强对偶 + 可微 ⇒ KKT条件是充要条件
       
       互补松弛性: λᵢ·fᵢ(x*) = 0
       影子价格: λᵢ ≈ -∂p*/∂uᵢ
```
